# SCADA系统全流程图 - 精确到每个部件

## 一、系统启动流程

```
run.py (入口)
│
├── 1. 初始化基础设施
│   ├── 创建日志目录 logs/
│   ├── 创建数据目录 data/
│   ├── 创建导出目录 exports/
│   └── 配置日志系统 (logging)
│
├── 2. 初始化存储层
│   └── Database (存储层/database.py)
│       └── SQLite数据库连接
│
├── 3. 初始化设备管理层
│   ├── 判断模式: --real → 真实设备 | 默认 → 模拟设备
│   ├── 模拟模式: SimulatedDeviceManager (采集层/simulated_device_manager.py)
│   │   └── 加载 配置/devices_simulated.yaml
│   └── 真实模式: RealDeviceManager (采集层/real_device_manager.py)
│       └── 加载 配置/devices_real.yaml
│
├── 4. 初始化报警系统
│   ├── AlarmOutput (报警层/alarm_output.py)
│   │   └── 声光报警器配置
│   ├── BroadcastSystem (报警层/broadcast_system.py)
│   │   └── 语音广播系统
│   └── AlarmManager (报警层/alarm_manager.py)
│       └── 加载 配置/alarms.yaml
│
├── 5. 初始化工业4.0智能层 ← 关键模块
│   ├── PredictiveMaintenance (智能层/predictive_maintenance.py)
│   ├── OEECalculator (智能层/oee_calculator.py)
│   ├── SPCAnalyzer (智能层/spc_analyzer.py)
│   ├── EnergyManager (智能层/energy_manager.py)
│   ├── EdgeDecisionEngine (智能层/edge_decision.py)
│   └── DeviceControlSafety (智能层/device_control.py)
│
├── 6. 初始化数据采集器
│   └── DataCollector (采集层/data_collector.py)
│       └── 注入所有智能层模块
│
├── 7. 创建Web应用
│   └── create_app() (展示层/routes.py)
│       ├── 注册API蓝图 (展示层/api/)
│       ├── 初始化WebSocket
│       └── 存储所有模块实例到 app 对象
│
├── 8. 连接设备
│   └── device_manager.connect_all()
│
├── 9. 启动数据采集
│   └── data_collector.start()
│
├── 10. 启动智能层模块
│   ├── predictive_maintenance.start()
│   ├── oee_calculator.start()
│   ├── energy_manager.start()
│   └── edge_decision.start()
│
├── 11. 模拟模式初始化 ← 问题可能在这里
│   └── SimulationInitializer (core/simulation_initializer.py)
│       ├── _init_oee_data()
│       ├── _init_energy_data()
│       ├── _init_predictive_maintenance_data()
│       └── _init_spc_data()
│
└── 12. 启动Web服务
    └── socketio.run(app, host='0.0.0.0', port=5000)
```

## 二、数据采集流程（实时运行）

```
DataCollector (采集层/data_collector.py)
│
├── 定时任务 (每2秒)
│   │
│   ├── 1. 读取设备数据
│   │   └── device_manager.read_all_devices()
│   │       ├── 模拟模式: 生成随机数据
│   │       └── 真实模式: Modbus通信读取
│   │
│   ├── 2. 存储到数据库
│   │   └── database.insert_data(device_id, register_name, value, timestamp)
│   │
│   ├── 3. 喂入智能层模块
│   │   ├── predictive_maintenance.feed_data(device_id, register_name, value, timestamp)
│   │   │   └── 更新健康评分
│   │   │
│   │   ├── oee_calculator.update_device_state(device_id, status)
│   │   │   └── 更新设备运行状态
│   │   │
│   │   ├── oee_calculator.record_production(device_id, count, good_count)
│   │   │   └── 记录产量
│   │   │
│   │   ├── spc_analyzer.feed_data(device_id, register_name, value)
│   │   │   └── SPC分析
│   │   │
│   │   ├── energy_manager.feed_power_data(device_id, power_kw)
│   │   │   └── 更新能耗数据
│   │   │
│   │   └── edge_decision.evaluate(device_id, data)
│   │       └── 边缘决策
│   │
│   └── 4. 检查报警
│       └── alarm_manager.check_alarms(device_id, data)
│
└── WebSocket推送
    └── socketio.emit('data_update', data)
```

## 四、核心模块内部数据结构

### 4.1 PredictiveMaintenance (智能层/predictive_maintenance.py)

```
内部数据结构:
├── health_scores: dict[str, dict]
│   └── key: "device_id:register_name"
│   └── value: {
│       "device_id": str,
│       "register_name": str,
│       "health_score": float (0-100),
│       "trend": "improving" | "stable" | "degrading",
│       "anomaly_count": int,
│       "last_analysis": datetime
│   }
│
├── time_series: dict[str, list]
│   └── key: "device_id:register_name"
│   └── value: [(timestamp, value), ...]
│
├── maintenance_alerts: list[dict]
│   └── [{severity, device_id, register_name, message, timestamp}, ...]
│
└── 方法:
    ├── feed_data(device_id, register_name, value, timestamp)
    │   └── 添加数据点，触发分析
    ├── get_health_scores()
    │   └── 返回所有设备健康评分
    ├── get_device_health(device_id)
    │   └── 返回指定设备健康评分
    ├── get_maintenance_alerts(limit)
    │   └── 返回维护建议列表
    └── _run_analysis()
        └── 执行健康分析算法
```

### 4.2 OEECalculator (智能层/oee_calculator.py)

```
内部数据结构:
├── shift_data: dict[str, dict]
│   └── key: device_id
│   └── value: {
│       "shift_start": datetime,
│       "planned_production_time": float (秒),
│       "actual_run_time": float (秒),
│       "downtime": float (秒),
│       "total_count": int,
│       "good_count": int,
│       "ideal_cycle_time": float (秒/件)
│   }
│
├── theoretical_rates: dict[str, float]
│   └── key: device_id
│   └── value: 件/小时
│
├── device_states: dict[str, dict]
│   └── key: device_id
│   └── value: {"status": str, "since": datetime}
│
└── 方法:
    ├── start_shift(device_id, planned_hours)
    │   └── 开始新班次
    ├── set_theoretical_rate(device_id, rate)
    │   └── 设置理论产能
    ├── update_device_state(device_id, status)
    │   └── 更新设备状态 (running/stopped/fault/idle)
    ├── record_production(device_id, count, good_count)
    │   └── 记录产量
    ├── calculate_oee(device_id)
    │   └── 计算OEE = 可用性 × 性能 × 质量
    └── get_all_oee()
        └── 返回所有设备OEE
```

### 4.3 SPCAnalyzer (智能层/spc_analyzer.py)

```
内部数据结构:
├── data_buffers: dict[str, list]
│   └── key: "device_id:register_name"
│   └── value: [value1, value2, ...]
│
├── spec_limits: dict[str, dict]
│   └── key: "device_id:register_name"
│   └── value: {"usl": float, "lsl": float, "target": float}
│
├── violations: dict[str, list]
│   └── key: "device_id:register_name"
│   └── value: [{rule, severity, description, timestamp}, ...]
│
└── 方法:
    ├── feed_data(device_id, register_name, value)
    │   └── 添加数据点，触发SPC分析
    ├── set_spec_limits(key, usl, lsl, target)
    │   └── 设置规格限
    ├── get_control_chart(device_id, register_name)
    │   └── 返回控制图数据 (X-bar R图)
    ├── get_capability(device_id, register_name)
    │   └── 返回过程能力 (Cp, Cpk)
    └── get_violations(device_id, limit)
        └── 返回判异结果
```

### 4.4 EnergyManager (智能层/energy_manager.py)

```
内部数据结构:
├── realtime_power: dict[str, dict]
│   └── key: device_id
│   └── value: {"power_kw": float, "timestamp": datetime}
│
├── energy_accumulated: dict[str, dict]
│   └── key: device_id
│   └── value: {
│       "energy_kwh": float,
│       "peak_kwh": float,
│       "flat_kwh": float,
│       "valley_kwh": float,
│       "water_m3": float,
│       "gas_m3": float,
│       "steam_ton": float
│   }
│
├── energy_baseline: dict[str, float]
│   └── key: device_id
│   └── value: 日均能耗 (kWh/天)

├── tariff: dict

│
└── 方法:
    ├── feed_power_data(device_id, power_kw, energy_kwh, timestamp)
    │   └── 喂入电力数据
    ├── set_baseline(device_id, daily_kwh)
    │   └── 设置能耗基线
    ├── get_realtime_power(device_id)
    │   └── 返回实时功率
    ├── get_total_power()
    │   └── 返回全厂总功率
    ├── get_energy_summary(device_id)
    │   └── 返回能耗汇总
    ├── get_energy_cost_breakdown()
    │   └── 返回分时电费明细
    └── get_carbon_emission()
        └── 返回碳排放数据
```
