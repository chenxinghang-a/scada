# 硬件选型说明

## 1 系统硬件总体方案

本系统采用"传感器+PLC+上位机"三层硬件架构，实现工业现场数据的采集、传输和监控。

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   传感器层    │ ──→ │   控制层     │ ──→ │   监控层     │
│ 温度/压力/电力 │     │  PLC/RTU    │     │  上位机(PC)  │
└─────────────┘     └─────────────┘     └─────────────┘
```

---

## 2 传感器选型

### 2.1 温度传感器

**选型：PT100铂电阻温度传感器 + 温度变送器**

| 参数 | 规格 |
|------|------|
| 型号 | WZP-PT100 |
| 测量范围 | -200°C ~ 650°C |
| 精度等级 | A级（±0.15°C @ 0°C） |
| 输出信号 | 4-20mA / RS485 Modbus |
| 响应时间 | ≤15s |
| 防护等级 | IP65 |
| 供电电压 | 24V DC |

**选型依据：**
- PT100是工业现场最常用的温度传感器，精度高、稳定性好
- 配套温度变送器将电阻信号转换为标准4-20mA或数字信号
- 支持Modbus RTU协议，可直接接入本系统

### 2.2 压力传感器

**选型：扩散硅压力变送器**

| 参数 | 规格 |
|------|------|
| 型号 | MPM489 |
| 测量范围 | 0 ~ 1.0 MPa |
| 精度 | ±0.25% FS |
| 输出信号 | 4-20mA / RS485 Modbus |
| 过载能力 | 2倍量程 |
| 供电电压 | 24V DC |
| 介质温度 | -40°C ~ 85°C |

**选型依据：**
- 扩散硅传感器适用于气体和液体压力测量
- 精度满足工业监控要求
- 支持Modbus通信，便于系统集成

### 2.3 电力仪表

**选型：多功能电力仪表**

| 参数 | 规格 |
|------|------|
| 型号 | APM810 |
| 测量参数 | 电压、电流、功率、电量、功率因数 |
| 电压范围 | AC 3×57.7/100V ~ 3×380/660V |
| 电流范围 | AC 0 ~ 6000A（经互感器） |
| 精度 | 0.5级 |
| 通信接口 | RS485 Modbus RTU |
| 显示 | LCD大屏幕显示 |

**选型依据：**
- 多功能仪表可同时测量多个电力参数
- 支持Modbus RTU协议，可直接接入系统
- 带显示功能，便于现场查看

## 3 控制器选型

### 3.1 PLC选型

**选型：西门子S7-1200 PLC**

| 参数 | 规格 |
|------|------|
| 型号 | 6ES7 214-1AG40-0XB0 |
| CPU | 1214C DC/DC/DC |
| 数字量输入 | 14点 |
| 数字量输出 | 10点 |
| 模拟量输入 | 2路 |
| 通信接口 | 以太网（Profinet）/ RS485 |
| 程序存储 | 100KB |
| 数据存储 | 4MB |

**选型依据：**
- 西门子S7-1200是中小型自动化项目的主流选择
- 支持Modbus TCP/RTU协议
- 编程环境TIA Portal成熟可靠
- 性价比高，适合毕设项目

### 3.2 通信模块

**选型：CM1241 RS485通信模块**

| 参数 | 规格 |
|------|------|
| 型号 | 6ES7 241-1CH32-0XB0 |
| 接口类型 | RS485 |
| 波特率 | 300 ~ 115200 bps |
| 协议 | Modbus RTU Master/Slave |

**选型依据：**
- 与S7-1200配套使用
- 支持Modbus RTU主站功能
- 可连接多个Modbus从站设备
