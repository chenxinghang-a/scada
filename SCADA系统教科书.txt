第一部分：系统概述与架构
========================

第1章：系统简介
--------------

### 1.1 系统功能概述

本系统是一个工业数据采集与监控系统（SCADA），主要功能包括：

1. **数据采集**：支持Modbus TCP、OPC UA、MQTT、REST等多种工业协议
2. **实时监控**：WebSocket实时数据推送，设备状态监控
3. **报警管理**：多级报警、声光报警器、语音广播
4. **数据分析**：OEE设备综合效率、SPC统计过程控制、预测性维护
5. **能源管理**：峰谷平电价、碳排放计算
6. **安全控制**：安全联锁、紧急停机、操作审计
7. **用户管理**：JWT认证、角色权限、操作日志

### 1.2 技术栈

- **编程语言**：Python 3.8+
- **Web框架**：Flask + Flask-SocketIO
- **数据库**：SQLite（主存储）+ TDengine（时序数据，可选）
- **前端技术**：HTML5 + CSS3 + JavaScript + Bootstrap 5
- **通信协议**：Modbus TCP、OPC UA、MQTT、REST
- **实时通信**：WebSocket
- **日志系统**：Python logging模块
- **配置管理**：YAML格式配置文件

### 1.3 系统特点

1. **模块化设计**：分层架构，职责清晰
2. **多协议支持**：兼容主流工业通信协议
3. **实时监控**：WebSocket实时数据推送
4. **智能分析**：内置多种工业分析算法
5. **安全可靠**：完善的权限控制和操作日志
6. **易于扩展**：支持自定义设备和协议

### 2.1 架构图

```
┌─────────────────────────────────────────────────────────────┐
```

### 4.2 完整源代码

```python
"""
工业数据采集与监控系统启动脚本
"""

import sys
import math
import time
import logging
import threading
from pathlib import Path
from datetime import datetime

# 添加项目根目录到Python路径
project_root = Path(__file__).parent
sys.path.insert(0, str(project_root))

# 记录系统启动时间
SYSTEM_START_TIME = datetime.now()

# 确保日志目录存在
(project_root / 'logs').mkdir(parents=True, exist_ok=True)
(project_root / 'data').mkdir(parents=True, exist_ok=True)
(project_root / 'exports').mkdir(parents=True, exist_ok=True)

# 配置日志
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.StreamHandler(),
        logging.FileHandler('logs/scada.log', encoding='utf-8')
    ]
)

logger = logging.getLogger(__name__)


def main():
    """主函数"""
    try:
        logger.info("正在启动工业数据采集与监控系统...")

        # 导入模块
        from 存储层.database import Database
        from 采集层.device_manager import DeviceManager
        from 采集层.real_device_manager import RealDeviceManager
        from 采集层.simulated_device_manager import SimulatedDeviceManager
        from 报警层.alarm_manager import AlarmManager
        from 采集层.data_collector import DataCollector
        from 展示层.routes import create_app

        # 判断是否使用模拟模式
        simulation_mode = '--real' not in sys.argv

        # 根据模式选择配置文件和数据库
        if simulation_mode:
            config_path = '配置/devices_simulated.yaml'
            db_path = 'data/scada_simulated.db'
            logger.info("模拟模式：使用模拟设备配置")
        else:
            config_path = '配置/devices_real.yaml'
            db_path = 'data/scada_real.db'
            logger.info("真实设备模式：使用真实设备配置")

        # 初始化数据库
        logger.info("初始化数据库...")
        database = Database(str(project_root / db_path))

        # 初始化设备管理器（根据模式选择不同的管理器）
        logger.info("加载设备配置...")
        if simulation_mode:
            device_manager = SimulatedDeviceManager(config_path)
        else:
            device_manager = RealDeviceManager(config_path)

        # 初始化报警管理器（含声光报警器+广播系统）
        logger.info("加载报警配置...")
        from 报警层.alarm_output import AlarmOutput
        from 报警层.broadcast_system import BroadcastSystem

        # 从alarms.yaml读取灯控设备配置
        import yaml
        alarms_config_path = Path('配置/alarms.yaml')
        alarm_output_cfg = {'enabled': True, 'simulation': simulation_mode}
        if alarms_config_path.exists():
            with open(alarms_config_path, 'r', encoding='utf-8') as f:
                alarms_yaml = yaml.safe_load(f) or {}
            ao_cfg = alarms_yaml.get('alarm_output', {})
            if ao_cfg:
                # 使用Patlite信号灯塔作为主输出
                tower = ao_cfg.get('signal_tower', {})
                alarm_output_cfg.update({
                    'modbus': {
                        'host': tower.get('host', '192.168.1.70'),
                        'port': tower.get('port', 502),
                        'slave_id': tower.get('slave_id', 1),
                    },
                    'do_mapping': tower.get('do_mapping', {
                        'red_light': 0, 'yellow_light': 1,
                        'green_light': 2, 'buzzer': 5
                    }),
                })
                logger.info(f"灯控设备: {tower.get('device_id')} @ {tower.get('host')}")
        alarm_output = AlarmOutput(alarm_output_cfg)
        broadcast_system = BroadcastSystem({
            'enabled': True,
            'simulation': simulation_mode,
            'areas': ['车间A', '车间B', '仓库', '办公楼'],
            'default_area': 'all',
            'preset_templates': {
                'alarm_critical': '注意！{area}发生严重报警：{message}，请立即处置！',
                'alarm_warning': '提醒：{area}出现告警：{message}，请关注。',
                'evacuation': '请注意，{area}发生紧急状况，请沿疏散通道撤离！',
                'all_clear': '广播通知，{area}警报解除，恢复正常。',
            }
        })
        alarm_manager = AlarmManager(database, '配置/alarms.yaml',
                                     alarm_output=alarm_output,
                                     broadcast_system=broadcast_system)

        # 初始化数据采集器（注入智能层模块）
        logger.info("初始化数据采集器...")

        # 初始化工业4.0智能层
        logger.info("初始化智能层模块...")
        from 智能层.predictive_maintenance import PredictiveMaintenance
        from 智能层.oee_calculator import OEECalculator
        from 智能层.spc_analyzer import SPCAnalyzer
        from 智能层.energy_manager import EnergyManager
        from 智能层.edge_decision import EdgeDecisionEngine
        from 智能层.device_control import DeviceControlSafety

        predictive_maintenance = PredictiveMaintenance(database)
        oee_calculator = OEECalculator(database)
        spc_analyzer = SPCAnalyzer(database)
        energy_manager = EnergyManager(database)
        edge_decision = EdgeDecisionEngine(database)
        device_control = DeviceControlSafety(database, device_manager, alarm_manager)

        # 初始化TDengine适配器（可选，需要TDengine服务）
        realtime_bridge = None
        tsdb_adapter = None
        try:
            from 智能层.tsdb_adapter import TSDBAdapter, RealtimeDataBridge
            from timeseries.tdengine_client import TDengineClient

            tdengine = TDengineClient("localhost", 6041)
            if tdengine.connect():
                tdengine.init_tables()

                # 创建实时数据桥接器
                realtime_bridge = RealtimeDataBridge(tdengine)

                # 创建智能层适配器
                tsdb_adapter = TSDBAdapter(
                    tdengine,
                    oee_calculator=oee_calculator,
                    predictive_maintenance=predictive_maintenance,
                    spc_analyzer=spc_analyzer,
                    energy_manager=energy_manager
                )

                logger.info("TDengine适配器初始化成功")
            else:
                logger.warning("TDengine连接失败，跳过TDengine集成")
        except Exception as e:
            logger.warning(f"TDengine初始化失败（可选组件）: {e}")

        # 注册边缘决策的动作回调
        def edge_set_alarm(message, level='warning'):
            """边缘决策报警回调"""
            from datetime import datetime
            logger.warning(f"[边缘决策报警] [{level.upper()}] {message}")
            # 记录到数据库
            try:
                database.insert_alarm(
                    alarm_id=f'edge_{int(datetime.now().timestamp())}',
                    device_id='edge_decision',
                    register_name='auto',
                    alarm_level=level,
                    alarm_message=message,
                    threshold=0,
                    actual_value=0,
                    timestamp=datetime.now()
                )
            except Exception as e:
                logger.error(f"边缘决策报警记录失败: {e}")

        edge_decision.register_action('set_alarm', edge_set_alarm)

        data_collector = DataCollector(
            device_manager, database, alarm_manager,
            predictive_maintenance=predictive_maintenance,
            oee_calculator=oee_calculator,
            spc_analyzer=spc_analyzer,
            energy_manager=energy_manager,
            edge_decision=edge_decision,
            device_control=device_control,
            realtime_bridge=realtime_bridge,
        )

        # 创建Flask应用
        logger.info("创建Web应用...")
        app = create_app(database, device_manager, alarm_manager, data_collector,
                         predictive_maintenance=predictive_maintenance,
                         oee_calculator=oee_calculator,
                         spc_analyzer=spc_analyzer,
                         energy_manager=energy_manager,
                         edge_decision=edge_decision,
                         device_control=device_control)

        # 将启动时间传递给app
        app.system_start_time = SYSTEM_START_TIME

        # ---- 后台连接设备 + 启动采集（不阻塞Web服务启动） ----
        def _background_start():
            """后台线程：连接设备 → 启动采集"""
            # 连接所有设备（带20s超时，避免阻塞太久）
            logger.info("后台连接设备...")
            connection_results = device_manager.connect_all(timeout=20)
            for device_id, success in connection_results.items():
                status = "成功" if success else "失败"
                logger.info(f"  设备 {device_id}: {status}")

            # 启动数据采集
            logger.info("启动数据采集...")
            data_collector.start()

            # 启动智能层模块
            logger.info("启动智能层模块...")
            predictive_maintenance.start()
            oee_calculator.start()
            energy_manager.start()
            edge_decision.start()

            # 启动TDengine适配器（如果可用）
            if realtime_bridge:
                realtime_bridge.start()
                logger.info("TDengine实时数据桥接器已启动")
            if tsdb_adapter:
                devices = device_manager.get_all_devices()
                for dev_id, dev_config in devices.items():
                    registers = dev_config.get('registers', [])
                    if registers:
                        tsdb_adapter.register_device(dev_id, registers)
                tsdb_adapter.start()
                logger.info("TDengine智能层适配器已启动")

            # 注入WebSocket推送函数到报警管理器
            from 展示层.websocket import emit_alarm, emit_broadcast
            alarm_manager.set_websocket_emit(emit_alarm)
            broadcast_system.add_callback(lambda msg: emit_broadcast(msg))

            logger.info("后台采集服务已就绪")

        bg_thread = threading.Thread(target=_background_start, daemon=True, name="bg_device_connect")
        bg_thread.start()

        # ---- 模拟模式初始化器（主线程创建，不依赖后台连接） ----
        simulation_initializer = None
        if simulation_mode:
            logger.info("模拟模式：初始化工业4.0模块数据...")
            from core.simulation_initializer import initialize_simulation_data
            simulation_initializer = initialize_simulation_data(
                device_manager, oee_calculator, predictive_maintenance,
                spc_analyzer, energy_manager
            )
            app.simulation_initializer = simulation_initializer

        # 立即启动Web服务（不等待设备连接）

        # 启动Web服务
        mode_str = "真实设备模式" if not simulation_mode else "模拟模式：使用仿真数据"
        logger.info("启动Web服务...")
        logger.info("=" * 50)
        logger.info("工业数据采集与监控系统已启动")
        logger.info("访问地址: http://localhost:5000")
        logger.info(f"（{mode_str}）")
        logger.info("=" * 50)

        # 使用routes.py中已创建的SocketIO实例（init_socketio在create_app中已调用）
        from 展示层.websocket import socketio
        from config import WebConfig
        host = WebConfig.HOST
        port = WebConfig.REAL_PORT if not simulation_mode else WebConfig.PORT
        if socketio is None:
            logger.warning("SocketIO未初始化，使用普通Flask服务器")
            app.run(host=host, port=port, debug=False)
        else:
            socketio.run(app, host=host, port=port, debug=False, allow_unsafe_werkzeug=True)

    except KeyboardInterrupt:
        logger.info("系统正在关闭...")
        if 'data_collector' in locals():
            data_collector.stop()
        if 'tsdb_adapter' in locals() and tsdb_adapter:
            tsdb_adapter.stop()
        if 'realtime_bridge' in locals() and realtime_bridge:
            realtime_bridge.stop()
        if 'alarm_output' in locals():
            alarm_output.disconnect()
        if 'broadcast_system' in locals():
            broadcast_system.disconnect()
        if 'device_manager' in locals():
            device_manager.disconnect_all()
        logger.info("系统已关闭")

    except Exception as e:
        logger.error(f"系统启动失败: {e}", exc_info=True)
        sys.exit(1)


if __name__ == '__main__':
    main()
```

#### 4.3.4 后台启动

```python
def _background_start():
    """后台线程：连接设备 → 启动采集"""
    # 连接所有设备（带20s超时，避免阻塞太久）
    connection_results = device_manager.connect_all(timeout=20)

    data_collector.start()
    
    # 启动智能层模块
    predictive_maintenance.start()
    oee_calculator.start()
    energy_manager.start()
    edge_decision.start()

bg_thread = threading.Thread(target=_background_start, daemon=True, name="bg_device_connect")
bg_thread.start()
```

**设计思想**：使用后台线程连接设备，避免阻塞Web服务启动。这样用户可以立即访问Web界面，而设备连接在后台进行。

### 6.1 模块概述

### 6.2 完整源代码

```python
"""
系统工厂模块（备用）
根据配置创建独立的模拟或真实组件

注意：当前run.py未使用此模块，直接在main()中创建组件。
此模块保留供run_v2.py或未来重构使用。
"""

import logging
from typing import Any

logger = logging.getLogger(__name__)


class SystemFactory:
    """
    系统工厂
    
    根据配置创建独立的模拟或真实组件，实现完全分离。
    
    使用方式：
        factory = SystemFactory(mode='simulated')  # 或 'real'
        device_manager = factory.create_device_manager()
        alarm_output = factory.create_alarm_output()
        broadcast_system = factory.create_broadcast_system()
    """

    def __init__(self, mode: str = 'simulated', config: dict[str, Any] | None = None):
        """
        初始化系统工厂
        
        Args:
            mode: 运行模式 ('simulated' 或 'real')
            config: 配置字典
        """
        if mode not in ('simulated', 'real'):
            raise ValueError(f"不支持的模式: {mode}，必须是 'simulated' 或 'real'")
        
        self.mode = mode
        self.config = config or {}
        
        logger.info(f"系统工厂初始化完成，模式: {mode}")

    def create_device_manager(self, config_path: str = None):
        """
        创建设备管理器
        
        Args:
            config_path: 设备配置文件路径
            
        Returns:
            设备管理器实例
        """
        if self.mode == 'simulated':
            from 采集层.simulated_device_manager import SimulatedDeviceManager
            logger.info("创建模拟设备管理器")
            return SimulatedDeviceManager(config_path)
        else:
            from 采集层.real_device_manager import RealDeviceManager
            logger.info("创建真实设备管理器")
            return RealDeviceManager(config_path)

    def create_alarm_output(self, config: dict[str, Any] = None):
        """
        创建报警输出
        
        Args:
            config: 报警输出配置
            
        Returns:
            报警输出实例
        """
        if self.mode == 'simulated':
            from 报警层.simulated_alarm_output import SimulatedAlarmOutput
            logger.info("创建模拟报警输出")
            return SimulatedAlarmOutput(config)
        else:
            from 报警层.real_alarm_output import RealAlarmOutput
            logger.info("创建真实报警输出")
            return RealAlarmOutput(config)

    def create_broadcast_system(self, config: dict[str, Any] = None):
        """
        创建广播系统
        
        Args:
            config: 广播系统配置
            
        Returns:
            广播系统实例
        """
        if self.mode == 'simulated':
            from 报警层.simulated_broadcast import SimulatedBroadcastSystem
            logger.info("创建模拟广播系统")
            return SimulatedBroadcastSystem(config)
        else:
            from 报警层.real_broadcast import RealBroadcastSystem
            logger.info("创建真实广播系统")
            return RealBroadcastSystem(config)

    def get_mode(self) -> str:
        """获取当前模式"""
        return self.mode

    def is_simulated(self) -> bool:
        """是否是模拟模式"""
        return self.mode == 'simulated'

    def is_real(self) -> bool:
        """是否是真实模式"""
        return self.mode == 'real'
```

### 7.1 模块概述

### 7.2 完整源代码

由于设备行为模拟器代码较长（1,000行），这里展示核心部分：

```python
"""
设备行为模拟器
================
基于物理模型和状态机的真实工业设备模拟

核心特性：
1. 物理模型 - 设备参数之间的关联性（温度→压力→流量）
2. 设备状态机 - 运行/空闲/故障/维护/停机
3. 故障模拟 - 真实的故障场景和渐进式退化
4. 数据连续性 - 确保数据流连续，支持历史回放
5. 班次影响 - 白班/夜班对设备参数的影响

参考：
- ISA-95 设备状态模型
- OEE六大损失理论
- 预测性维护最佳实践
"""

import math
import time
import random
import logging
import threading
from enum import Enum
from datetime import datetime, timedelta
from typing import Any, Dict, List, Optional, Callable
from dataclasses import dataclass, field
from collections import deque

logger = logging.getLogger(__name__)


class DeviceState(Enum):
    """设备状态（ISA-95标准）"""
    STOPPED = 0      # 停机
    IDLE = 1         # 空闲/待机
    RUNNING = 2      # 运行
    FAULT = 3        # 故障
    MAINTENANCE = 4  # 维护中
    SETUP = 5        # 换型/调试


class FaultType(Enum):
    """故障类型"""
    NONE = "none"
    SENSOR_DRIFT = "sensor_drift"        # 传感器漂移
    OVERHEATING = "overheating"          # 过热
    PRESSURE_LEAK = "pressure_leak"      # 压力泄漏
    MOTOR_WEAR = "motor_wear"            # 电机磨损
    COMMUNICATION = "communication"      # 通信故障
    POWER_FLUCTUATION = "power_fluctuation"  # 电源波动


@dataclass
class DeviceHealth:
    """设备健康状态"""
    overall_score: float = 100.0        # 总体健康评分 0-100
    mechanical_health: float = 100.0    # 机械健康
    electrical_health: float = 100.0    # 电气健康
    thermal_health: float = 100.0       # 热健康
    vibration_health: float = 100.0     # 振动健康
    last_maintenance: datetime = field(default_factory=datetime.now)
    operating_hours: float = 0.0        # 累计运行小时
    cycle_count: int = 0                # 累计循环次数


@dataclass
class ProcessModel:
    """过程模型参数"""
    # 基础参数
    base_temperature: float = 50.0      # 基础温度
    base_pressure: float = 0.8          # 基础压力
    base_flow: float = 15.0             # 基础流量
    base_level: float = 60.0            # 基础液位
    
    # 关联系数（温度升高→压力升高→流量变化）
    temp_pressure_coeff: float = 0.005  # 温度对压力的影响系数
    temp_flow_coeff: float = 0.02       # 温度对流量的影响系数
    pressure_flow_coeff: float = 0.1    # 压力对流量的影响系数
    
    # 时间常数（模拟惯性）
    thermal_time_constant: float = 120.0    # 热惯性时间常数（秒）
    pressure_time_constant: float = 30.0    # 压力响应时间常数
    flow_time_constant: float = 15.0        # 流量响应时间常数


class DeviceBehaviorSimulator:
    """
    设备行为模拟器
    
    模拟真实工业设备的行为，包括：
    - 物理参数关联
    - 状态转换
    - 故障注入
    - 班次影响
    - 写操作反馈（写入影响读取）
    - 设备特定输出（只生成该设备有的参数）
    """
    
    def __init__(self, device_id: str, device_config: Dict[str, Any]):
        """
        初始化设备行为模拟器
        
        Args:
            device_id: 设备唯一标识
            device_config: 设备配置字典，包含寄存器定义等
        """
        self.device_id = device_id
        self.device_config = device_config
        
        # 设备状态
        self.state = DeviceState.IDLE
        self.state_start_time = datetime.now()
        
        # 物理模型参数
        self.process_model = ProcessModel()
        
        # 当前物理参数值
        self.current_temperature = self.process_model.base_temperature
        self.current_pressure = self.process_model.base_pressure
        self.current_flow = self.process_model.base_flow
        self.current_level = self.process_model.base_level
        
        # 设备健康状态
        self.health = DeviceHealth()
        
        # 故障注入参数
        self.fault_type = FaultType.NONE
        self.fault_intensity = 0.0
        self.fault_start_time = None
        
        # 班次影响
        self.shift_factor = 1.0  # 白班1.0，夜班0.95
        
        # 历史数据缓存（用于趋势分析）
        self.history_buffer = deque(maxlen=1000)
        
        # 线程锁（保证线程安全）
        self._lock = threading.Lock()
        
        logger.info(f"设备 {device_id} 行为模拟器初始化完成")
    
    def _update_physical_parameters(self, dt: float):
        """
        更新物理参数（基于物理模型）
        
        这个方法实现了物理参数的动态更新，考虑了：
        1. 参数之间的关联性
        2. 时间常数（惯性）
        3. 随机噪声
        4. 故障影响
        
        Args:
            dt: 时间间隔（秒）
        """
        with self._lock:
            # 1. 计算目标值（基于当前状态和班次影响）
            target_temp = self.process_model.base_temperature * self.shift_factor
            target_pressure = self.process_model.base_pressure * self.shift_factor
            target_flow = self.process_model.base_flow * self.shift_factor
            
            # 2. 添加随机扰动（模拟真实环境）
            temp_noise = random.gauss(0, 0.5)
            pressure_noise = random.gauss(0, 0.02)
            flow_noise = random.gauss(0, 0.3)
            
            # 3. 应用物理关联（温度→压力→流量）
            # 温度变化影响压力
            temp_effect_on_pressure = (self.current_temperature - target_temp) * \
                                      self.process_model.temp_pressure_coeff
            target_pressure += temp_effect_on_pressure
            
            # 温度变化影响流量
            temp_effect_on_flow = (self.current_temperature - target_temp) * \
                                  self.process_model.temp_flow_coeff
            target_flow += temp_effect_on_flow
            
            # 压力变化影响流量
            pressure_effect_on_flow = (self.current_pressure - target_pressure) * \
                                      self.process_model.pressure_flow_coeff
            target_flow += pressure_effect_on_flow
            
            # 4. 应用时间常数（一阶惯性环节）
            # 公式: current = current + (target - current) * (1 - exp(-dt/tau))
            alpha_temp = 1 - math.exp(-dt / self.process_model.thermal_time_constant)
            alpha_pressure = 1 - math.exp(-dt / self.process_model.pressure_time_constant)
            alpha_flow = 1 - math.exp(-dt / self.process_model.flow_time_constant)
            
            self.current_temperature += (target_temp - self.current_temperature) * alpha_temp + temp_noise
            self.current_pressure += (target_pressure - self.current_pressure) * alpha_pressure + pressure_noise
            self.current_flow += (target_flow - self.current_flow) * alpha_flow + flow_noise
            
            # 5. 应用故障影响
            if self.fault_type != FaultType.NONE:
                self._apply_fault_effect()
            
            # 6. 限制参数范围（防止异常值）
            self.current_temperature = max(0, min(200, self.current_temperature))
            self.current_pressure = max(0, min(2, self.current_pressure))
            self.current_flow = max(0, min(50, self.current_flow))
            self.current_level = max(0, min(100, self.current_level))
```

第9章：数据采集器

### 9.1 模块概述

数据采集器是SCADA系统的"心脏"，负责定时从各种工业设备采集数据。

**核心职责**：
1. **定时采集**：按照配置的时间间隔，定期读取设备数据
2. **多协议支持**：支持Modbus TCP、OPC UA、MQTT、REST等多种协议
3. **数据队列**：采集到的数据放入队列，由专门的处理线程处理
4. **统计信息**：记录采集成功/失败次数、队列大小等

**生活比喻**：
数据采集器就像一个"快递员"，按照固定路线（设备列表）和时间表（采集间隔），定期去各个"收件点"（设备）取"包裹"（数据），然后送到"分拣中心"（数据处理线程）。

### 9.2 完整源代码

```python
"""
数据采集器模块
实现定时数据采集和数据处理
支持协议: Modbus TCP/RTU, OPC UA, MQTT, REST HTTP
"""

import time
import queue
import logging
import threading
from typing import Any
from datetime import datetime
from queue import Queue

logger = logging.getLogger(__name__)


class DataCollector:
    """
    数据采集器
    统一定时从多协议设备采集数据（Modbus/OPC UA/MQTT/REST）
    OPC UA和MQTT有自带的推送机制，仅对Modbus/REST做轮询
    """

    def __init__(self, device_manager, database, alarm_manager=None,
                 predictive_maintenance=None, oee_calculator=None,
                 spc_analyzer=None, energy_manager=None, edge_decision=None,
                 device_control=None, realtime_bridge=None):
        """
        初始化数据采集器

        Args:
            device_manager: 设备管理器实例，用于获取设备客户端
            database: 数据库实例，用于存储采集到的数据
            alarm_manager: 报警管理器实例（可选）
            predictive_maintenance: 预测性维护实例（可选）
            oee_calculator: OEE计算器实例（可选）
            spc_analyzer: SPC分析器实例（可选）
            energy_manager: 能源管理实例（可选）
            edge_decision: 边缘决策引擎实例（可选）
            device_control: 设备控制实例（可选）
            realtime_bridge: TDengine实时数据桥接器（可选）
        """
        self.device_manager = device_manager
        self.database = database
        self.alarm_manager = alarm_manager

        # 工业4.0智能层模块（可选注入）
        self.predictive_maintenance = predictive_maintenance
        self.oee_calculator = oee_calculator
        self.spc_analyzer = spc_analyzer
        self.energy_manager = energy_manager
        self.edge_decision = edge_decision
        self.device_control = device_control

        # TDengine实时数据桥接器（可选）
        self.realtime_bridge = realtime_bridge

        # 采集任务字典：device_id -> threading.Timer
        # 每个设备都有一个独立的定时器，按各自的频率采集
        self.tasks = {}  # device_id -> threading.Timer
        self.running = False

        # 数据队列：采集到的数据先放入队列，由专门的处理线程处理
        # 使用Queue实现线程安全的生产者-消费者模式
        self.data_queue = Queue(maxsize=10000)

        # 统计信息（用锁保护，多线程安全）
        self._stats_lock = threading.Lock()
        self.stats: dict[str, Any] = {
            'total_collections': 0,      # 总采集次数
            'successful_collections': 0, # 成功次数
            'failed_collections': 0,     # 失败次数
            'last_collection_time': None,# 最后采集时间
            'queue_size': 0,             # 队列大小
            'protocols_active': {}       # 各协议设备数量
        }

        # 数据处理线程
        self.process_thread = None

    def _inc_stat(self, key: str, amount: int = 1):
        """线程安全地增加统计计数"""
        with self._stats_lock:
            self.stats[key] = self.stats.get(key, 0) + amount

    def start(self):
        """
        启动数据采集

        工作流程：
        1. 启动数据处理线程（消费者）
        2. 遍历所有设备，根据协议类型启动采集任务
        3. OPC UA/MQTT使用推送模式（设备主动推送数据）
        4. Modbus/REST使用轮询模式（定时读取数据）
        """
        if self.running:
            logger.warning("数据采集器已在运行")
            return

        self.running = True

        # 启动数据处理线程（消费者线程）
        # daemon=True表示主线程退出时，这个线程也会自动退出
        self.process_thread = threading.Thread(target=self._process_data, daemon=True)
        self.process_thread.start()

        # 为每个设备启动采集任务
        devices = self.device_manager.get_all_devices()
        proto_count = {}
        for device_id, device_config in devices.items():
            if device_config.get('enabled', True):
                protocol = device_config.get('protocol', 'modbus_tcp')
                proto_count[protocol] = proto_count.get(protocol, 0) + 1

                # 根据协议类型选择采集模式
                if protocol in ('opcua', 'mqtt'):
                    # OPC UA和MQTT有自带的推送机制，设置回调即可
                    self._setup_push_device(device_id, device_config)
                else:
                    # Modbus和REST使用轮询模式，启动定时器
                    self._start_device_collection(device_id, device_config)

        # 更新统计信息
        with self._stats_lock:
            self.stats['protocols_active'] = proto_count

        summary = ', '.join(f"{k}:{v}" for k, v in proto_count.items())
        logger.info(f"数据采集器已启动，共 {len(devices)} 个设备 ({summary})")

    def stop(self):
        """停止数据采集"""
        self.running = False

        # 取消所有定时器
        for device_id, timer in self.tasks.items():
            timer.cancel()
        self.tasks.clear()

        # 断开所有设备连接
        self.device_manager.disconnect_all()
        logger.info("数据采集器已停止")

    def start_device_task(self, device_id: str, device_config: dict[str, Any]):
        """
        为指定设备启动采集任务（运行时添加设备时调用）

        设备存在 ⟹ 有采集任务 ⟹ 有数据
        设备不存在 ⟹ 无采集任务 ⟹ 无数据
        """
        if not self.running:
            logger.warning(f"数据采集器未运行，跳过设备 {device_id} 的采集启动")
            return

        if not device_config.get('enabled', True):
            logger.info(f"设备 {device_id} 已禁用，跳过采集启动")
            return

        if device_id in self.tasks:
            logger.debug(f"设备 {device_id} 已有采集任务，跳过")
            return

        protocol = device_config.get('protocol', 'modbus_tcp')
        if protocol in ('opcua', 'mqtt'):
            self._setup_push_device(device_id, device_config)
        else:
            self._start_device_collection(device_id, device_config)
        logger.info(f"已启动设备 {device_id} [{protocol}] 的采集任务")

    def remove_device_task(self, device_id: str):
        """
        删除设备的采集任务（设备被删除时调用）

        取消定时器，不再采集该设备数据。历史数据保留。
        """
        timer = self.tasks.pop(device_id, None)
        if timer:
            timer.cancel()
            logger.info(f"已停止设备 {device_id} 的采集任务")

    def _setup_push_device(self, device_id: str, device_config: dict[str, Any]):
        """
        设置推送型设备（OPC UA / MQTT）

        这些协议有自带的订阅/推送机制，不需要我们定时轮询。
        只需要设置回调函数，当设备推送数据时自动调用。
        """
        protocol = device_config.get('protocol', 'modbus_tcp')
        client = self.device_manager.get_client(device_id)
        if client is None:
            logger.error(f"设备 {device_id} 的客户端不存在")
            return

        # 设置数据回调函数
        # 当设备推送数据时，会自动调用这个函数
        def on_push_data(data):
            """推送数据回调"""
            if not self.running:
                return
            # 将数据放入队列
            self.data_queue.put({
                'device_id': device_id,
                'data': data,
                'timestamp': datetime.now(),
                'source': 'push'
            })

        # 根据协议类型设置回调
        if protocol == 'opcua':
            client.set_subscription_callback(on_push_data)
        elif protocol == 'mqtt':
            client.set_message_callback(on_push_data)

        logger.info(f"已设置 {device_id} [{protocol}] 的推送回调")

    def _start_device_collection(self, device_id: str, device_config: dict[str, Any]):
        """
        为轮询型设备启动采集定时器

        使用threading.Timer实现定时采集：
        1. 采集一次数据
        2. 设置下一次采集的定时器
        3. 重复直到系统停止
        """
        interval = device_config.get('collection_interval', 5)  # 默认5秒

        def collect_and_reschedule():
            """采集一次数据并重新设置定时器"""
            if not self.running:
                return

            # 采集数据
            self._collect_device_data(device_id)

            # 重新设置定时器（实现周期性采集）
            if self.running and device_id in self.tasks:
                timer = threading.Timer(interval, collect_and_reschedule)
                timer.daemon = True
                timer.start()
                self.tasks[device_id] = timer

        # 启动第一次采集
        timer = threading.Timer(interval, collect_and_reschedule)
        timer.daemon = True
        timer.start()
        self.tasks[device_id] = timer

    def _collect_device_data(self, device_id: str):
        """
        采集单个设备的数据

        工作流程：
        1. 获取设备客户端
        2. 调用客户端的read_all()方法读取所有寄存器
        3. 将数据放入队列
        4. 更新统计信息
        """
        try:
            # 获取设备客户端
            client = self.device_manager.get_client(device_id)
            if client is None:
                self._inc_stat('failed_collections')
                return

            # 读取所有寄存器数据
            data = client.read_all()
            if data is None:
                self._inc_stat('failed_collections')
                return

            # 将数据放入队列
            self.data_queue.put({
                'device_id': device_id,
                'data': data,
                'timestamp': datetime.now(),
                'source': 'poll'
            })

            # 更新统计信息
            self._inc_stat('total_collections')
            self._inc_stat('successful_collections')

        except Exception as e:
            logger.error(f"采集设备 {device_id} 数据失败: {e}")
            self._inc_stat('failed_collections')

    def _process_data(self):
        """
        数据处理线程（消费者）

        从队列中取出数据，进行以下处理：
        1. 存储到数据库（实时数据表 + 历史数据表）
        2. 检查报警规则
        3. 喂入智能层模块（OEE/SPC/预测性维护/能源管理）
        4. 推送到TDengine（如果配置了）
        """
        while self.running:
            try:
                # 从队列取数据，最多等待1秒
                item = self.data_queue.get(timeout=1)
                if item is None:
                    continue

                device_id = item['device_id']
                data = item['data']
                timestamp = item['timestamp']

                # 1. 存储到数据库
                self.database.save_data(device_id, data, timestamp)

                # 2. 检查报警
                if self.alarm_manager:
                    self.alarm_manager.check_alarms(device_id, data, timestamp)

                # 3. 喂入智能层模块
                self._feed_intelligent_modules(device_id, data, timestamp)

                # 4. 推送到TDengine
                if self.realtime_bridge:
                    self.realtime_bridge.push(device_id, data, timestamp)

                # 更新队列大小统计
                with self._stats_lock:
                    self.stats['queue_size'] = self.data_queue.qsize()

            except queue.Empty:
                # 队列为空，继续等待
                continue
            except Exception as e:
                logger.error(f"数据处理异常: {e}")

    def _feed_intelligent_modules(self, device_id: str, data: dict, timestamp: datetime):
        """
        将数据喂入智能层模块

        每个模块都需要数据来进行分析：
        - 预测性维护：分析设备健康趋势
        - OEE计算：统计设备效率
        - SPC分析：监控产品质量
        - 能源管理：统计能耗
        - 边缘决策：实时决策
        """
        # 预测性维护
        if self.predictive_maintenance:
            for register_name, value in data.items():
                if isinstance(value, (int, float)):
                    self.predictive_maintenance.feed_data(
                        device_id, register_name, value, timestamp
                    )

        # OEE计算器
        if self.oee_calculator:
            # OEE需要设备状态和产量数据
            device_status = data.get('status', 'unknown')
            self.oee_calculator.update_device_state(device_id, device_status)

        # SPC分析器
        if self.spc_analyzer:
            # SPC关注质量相关参数（如温度、压力、尺寸等）
            quality_params = ['temperature', 'pressure', 'thickness', 'ph']
            for param in quality_params:
                if param in data:
                    self.spc_analyzer.feed_data(device_id, param, data[param])

        # 能源管理
        if self.energy_manager:
            # 能源管理关注功率、电量等参数
            if 'power' in data:
                self.energy_manager.feed_power_data(device_id, data['power'], timestamp)

        # 边缘决策引擎
        if self.edge_decision:
            # 边缘决策需要所有实时数据
            self.edge_decision.feed_data(device_id, data, timestamp)

    def get_stats(self) -> dict[str, Any]:
        """获取采集统计信息"""
        with self._stats_lock:
            return dict(self.stats)
```

### 10.1 模块概述

**核心职责**：

**生活比喻**：
设备管理器就像一个"设备租赁公司"，你告诉它需要什么设备（配置文件），它就帮你找到对应的"司机"（协议客户端），并管理这些"司机"的工作状态。

### 10.2 完整源代码

```python
"""
设备管理器模块
管理所有设备的连接和配置
支持协议: Modbus TCP/RTU, OPC UA, MQTT, REST HTTP
面向现代黑灯工厂的多协议统一管理
"""

import logging
import yaml
import socket
from typing import Any
from pathlib import Path

logger = logging.getLogger(__name__)


# ---- 协议工厂：根据protocol字段自动选择客户端 ----
def _create_client(config: dict[str, Any], simulation_mode: bool, use_enhanced: bool = True):
    """
    根据协议类型创建对应的客户端实例

    模拟模式下所有协议都使用模拟客户端，无需真实设备。
    真实模式下按协议创建对应的真实客户端。

    Args:
        config: 设备配置字典
        simulation_mode: 是否模拟模式
        use_enhanced: 是否使用增强版模拟客户端（默认True）

    Returns:
        协议客户端实例
    """
    protocol = config.get('protocol', 'modbus_tcp')

    if protocol in ('modbus_tcp', 'modbus_rtu'):
        if simulation_mode:
            if use_enhanced:
                from .enhanced_simulated_client import EnhancedSimulatedModbusClient
                return EnhancedSimulatedModbusClient(config)
            else:
                from .simulated_client import SimulatedModbusClient
                return SimulatedModbusClient(config)
        else:
            from .modbus_client import ModbusClient
            return ModbusClient(config)

    elif protocol == 'opcua':
        if simulation_mode:
            if use_enhanced:
                from .enhanced_simulated_client import EnhancedSimulatedOPCUAClient
                return EnhancedSimulatedOPCUAClient(config)
            else:
                from .simulated_client import SimulatedOPCUAClient
                return SimulatedOPCUAClient(config)
        else:
            from .opcua_client import OPCUAClient
            return OPCUAClient(config)

    elif protocol == 'mqtt':
        if simulation_mode:
            if use_enhanced:
                from .enhanced_simulated_client import EnhancedSimulatedMQTTClient
                return EnhancedSimulatedMQTTClient(config)
            else:
                from .simulated_client import SimulatedMQTTClient
                return SimulatedMQTTClient(config)
        else:
            from .mqtt_client import MQTTClient
            return MQTTClient(config)

    elif protocol == 'rest':
        if simulation_mode:
            if use_enhanced:
                from .enhanced_simulated_client import EnhancedSimulatedRESTClient
                return EnhancedSimulatedRESTClient(config)
            else:
                from .simulated_client import SimulatedRESTClient
                return SimulatedRESTClient(config)
        else:
            from .rest_client import RESTDeviceClient
            return RESTDeviceClient(config)

    else:
        logger.error(f"不支持的协议类型: {protocol}")
        return None


class DeviceManager:
    """
    设备管理器
    统一管理 Modbus / OPC UA / MQTT / REST 四种协议设备

    黑灯工厂核心能力：
    1. 多协议统一接入 - 工厂内不同品牌/型号设备一网打尽
    2. 配置热加载 - 运行时新增/删除设备无需重启
    3. 模拟模式 - 无真实设备时自动切换模拟数据
    4. 自动重连 - 设备断开后自动恢复连接
    """

    # 支持的协议列表
    SUPPORTED_PROTOCOLS = ['modbus_tcp', 'modbus_rtu', 'opcua', 'mqtt', 'rest']

    def __init__(self, config_path: str | None = None, simulation_mode: bool = True, use_enhanced_simulation: bool = True):
        """
        初始化设备管理器

        Args:
            config_path: 设备配置文件路径
            simulation_mode: 是否启用模拟模式（无真实设备时使用模拟数据）
            use_enhanced_simulation: 是否使用增强版模拟（物理模型驱动）
        """
        self.config_path = config_path or '配置/devices.yaml'
        self.simulation_mode = simulation_mode
        self.use_enhanced_simulation = use_enhanced_simulation
        self.devices = {}  # device_id -> device_config
        self.clients = {}  # device_id -> protocol client

        # 加载设备配置
        self.load_config()

    def load_config(self):
        """加载设备配置文件"""
        try:
            config_file = Path(self.config_path)
            if not config_file.exists():
                logger.error(f"配置文件不存在: {self.config_path}")
                return

            with open(config_file, 'r', encoding='utf-8') as f:
                config = yaml.safe_load(f)

            # 解析设备配置
            devices_config = config.get('devices', [])
            for device_config in devices_config:
                device_id = device_config.get('id')
                if device_id:
                    self.devices[device_id] = device_config
                    protocol = device_config.get('protocol', 'modbus_tcp')
                    logger.info(f"加载设备配置: {device_id} [{protocol}] - {device_config.get('name')}")

            # 按协议统计
            proto_count = {}
            for d in self.devices.values():
                p = d.get('protocol', 'modbus_tcp')
                proto_count[p] = proto_count.get(p, 0) + 1
            summary = ', '.join(f"{k}:{v}" for k, v in proto_count.items())
            logger.info(f"共加载 {len(self.devices)} 个设备 ({summary})")

        except Exception as e:
            logger.error(f"加载配置文件异常: {e}")

    def connect_all(self):
        """
        连接所有设备

        遍历所有设备配置，为每个设备创建协议客户端并连接。
        模拟模式下，客户端会自动生成模拟数据。
        """
        for device_id, device_config in self.devices.items():
            try:
                # 使用协议工厂创建客户端
                client = _create_client(
                    device_config,
                    simulation_mode=self.simulation_mode,
                    use_enhanced=self.use_enhanced_simulation
                )
                if client is None:
                    logger.error(f"创建设备 {device_id} 的客户端失败")
                    continue

                # 连接设备
                if client.connect():
                    self.clients[device_id] = client
                    logger.info(f"设备 {device_id} 连接成功")
                else:
                    logger.error(f"设备 {device_id} 连接失败")

            except Exception as e:
                logger.error(f"连接设备 {device_id} 异常: {e}")

        logger.info(f"设备连接完成: {len(self.clients)}/{len(self.devices)} 成功")

    def disconnect_all(self):
        """断开所有设备连接"""
        for device_id, client in self.clients.items():
            try:
                client.disconnect()
                logger.info(f"设备 {device_id} 已断开")
            except Exception as e:
                logger.error(f"断开设备 {device_id} 异常: {e}")
        self.clients.clear()

    def get_client(self, device_id: str):
        """获取设备的协议客户端"""
        return self.clients.get(device_id)

    def get_all_devices(self) -> dict[str, Any]:
        """获取所有设备配置"""
        return dict(self.devices)

    def get_device_status(self, device_id: str) -> dict[str, Any]:
        """获取设备状态"""
        if device_id not in self.devices:
            return {'error': f'设备 {device_id} 不存在'}

        config = self.devices[device_id]
        client = self.clients.get(device_id)

        return {
            'device_id': device_id,
            'name': config.get('name'),
            'protocol': config.get('protocol'),
            'connected': client is not None and client.is_connected(),
            'enabled': config.get('enabled', True),
            'simulation': self.simulation_mode
        }

    def get_all_status(self) -> list[dict[str, Any]]:
        """获取所有设备状态"""
        statuses = []
        for device_id in self.devices:
            statuses.append(self.get_device_status(device_id))
        return statuses

    def add_device(self, device_config: dict[str, Any]) -> bool:
        """
        添加新设备（运行时热加载）

        Args:
            device_config: 设备配置字典

        Returns:
            是否添加成功
        """
        device_id = device_config.get('id')
        if not device_id:
            logger.error("设备配置缺少id字段")
            return False

        if device_id in self.devices:
            logger.warning(f"设备 {device_id} 已存在，将覆盖配置")

        # 保存配置
        self.devices[device_id] = device_config
        self._save_config()

        # 创建客户端并连接
        try:
            client = _create_client(
                device_config,
                simulation_mode=self.simulation_mode,
                use_enhanced=self.use_enhanced_simulation
            )
            if client and client.connect():
                self.clients[device_id] = client
                logger.info(f"新设备 {device_id} 添加成功并已连接")
                return True
        except Exception as e:
            logger.error(f"连接新设备 {device_id} 失败: {e}")

        return False

    def remove_device(self, device_id: str) -> bool:
        """
        删除设备（运行时热删除）

        Args:
            device_id: 设备ID

        Returns:
            是否删除成功
        """
        if device_id not in self.devices:
            logger.warning(f"设备 {device_id} 不存在")
            return False

        # 断开连接
        client = self.clients.pop(device_id, None)
        if client:
            try:
                client.disconnect()
            except Exception as e:
                logger.error(f"断开设备 {device_id} 失败: {e}")

        # 删除配置
        del self.devices[device_id]
        self._save_config()

        logger.info(f"设备 {device_id} 已删除")
        return True

    def _save_config(self):
        """保存设备配置到YAML文件"""
        try:
            config = {'devices': list(self.devices.values())}
            with open(self.config_path, 'w', encoding='utf-8') as f:
                yaml.dump(config, f, allow_unicode=True, default_flow_style=False)
            logger.info(f"设备配置已保存到 {self.config_path}")
        except Exception as e:
            logger.error(f"保存配置文件失败: {e}")
```

### 11.1 模块概述

数据库模块是SCADA系统的"记忆"，负责存储所有采集到的数据。

**核心职责**：
1. **数据存储**：实时数据、历史数据、报警记录、设备状态
2. **数据查询**：按时间范围、设备ID、寄存器名称查询
3. **数据清理**：清理过期数据，释放存储空间

**生活比喻**：
数据库就像一个"档案室"，有专门的"档案柜"（表），每个"档案柜"里有"文件夹"（索引），可以快速找到需要的"文件"（数据）。

### 11.2 完整源代码

```python
"""
数据库模块
实现SQLite数据库操作
"""

import sqlite3
import logging
from typing import Any
from datetime import datetime, timedelta
from pathlib import Path
from contextlib import contextmanager

logger = logging.getLogger(__name__)


class Database:
    """
    SQLite数据库管理类
    负责数据存储、查询和维护
    """

    def __init__(self, db_path: str = 'data/scada.db'):
        """
        初始化数据库

        Args:
            db_path: 数据库文件路径
        """
        self.db_path = db_path

        # 确保目录存在
        Path(db_path).parent.mkdir(parents=True, exist_ok=True)

        # 初始化数据库
        self._init_database()

    @contextmanager
    def get_connection(self):
        """
        获取数据库连接（上下文管理器）
        使用WAL模式提升并发性能，设置超时避免死锁

        Yields:
            sqlite3.Connection: 数据库连接
        """
        conn = sqlite3.connect(self.db_path, timeout=30)
        conn.row_factory = sqlite3.Row
        # WAL模式：允许多个读连接和一个写连接并发
        conn.execute('PRAGMA journal_mode=WAL')
        # 忙等待超时30秒
        conn.execute('PRAGMA busy_timeout=30000')
        try:
            yield conn
            conn.commit()
        except Exception as e:
            conn.rollback()
            raise e
        finally:
            conn.close()

    def _init_database(self):
        """初始化数据库表结构"""
        with self.get_connection() as conn:
            cursor = conn.cursor()

            # 创建实时数据表（每个设备+寄存器只保留最新一条）
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS realtime_data (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    device_id TEXT NOT NULL,
                    register_name TEXT NOT NULL,
                    value REAL,
                    unit TEXT,
                    timestamp DATETIME NOT NULL,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
                    UNIQUE(device_id, register_name)
                )
            ''')

            # 创建历史数据表
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS history_data (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    device_id TEXT NOT NULL,
                    register_name TEXT NOT NULL,
                    value REAL,
                    unit TEXT,
                    timestamp DATETIME NOT NULL,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')

            # 创建报警记录表
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS alarm_records (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    alarm_id TEXT NOT NULL,
                    device_id TEXT NOT NULL,
                    register_name TEXT NOT NULL,
                    alarm_level TEXT NOT NULL,
                    alarm_message TEXT,
                    threshold REAL,
                    actual_value REAL,
                    timestamp DATETIME NOT NULL,
                    acknowledged BOOLEAN DEFAULT 0,
                    acknowledged_at DATETIME,
                    acknowledged_by TEXT,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')

            # 创建设备状态表
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS device_status (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    device_id TEXT NOT NULL,
                    status TEXT NOT NULL,
                    message TEXT,
                    timestamp DATETIME NOT NULL,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')

            # 创建索引
            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_realtime_device_time 
                ON realtime_data(device_id, timestamp)
            ''')

            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_realtime_register 
                ON realtime_data(device_id, register_name, timestamp)
            ''')

            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_history_device_time 
                ON history_data(device_id, timestamp)
            ''')

            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_history_register 
                ON history_data(device_id, register_name, timestamp)
            ''')

            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_alarm_device_time 
                ON alarm_records(device_id, timestamp)
            ''')

            logger.info("数据库初始化完成")

    def save_data(self, device_id: str, data: dict[str, Any], timestamp: datetime):
        """
        保存设备数据（实时数据 + 历史数据）

        Args:
            device_id: 设备ID
            data: 数据字典 {register_name: value}
            timestamp: 时间戳
        """
        with self.get_connection() as conn:
            cursor = conn.cursor()

            for register_name, value in data.items():
                if not isinstance(value, (int, float)):
                    continue

                # 更新实时数据（使用UPSERT）
                cursor.execute('''
                    INSERT INTO realtime_data (device_id, register_name, value, timestamp)
                    VALUES (?, ?, ?, ?)
                    ON CONFLICT(device_id, register_name)
                    DO UPDATE SET value = ?, timestamp = ?
                ''', (device_id, register_name, value, timestamp,
                      value, timestamp))

                # 插入历史数据
                cursor.execute('''
                    INSERT INTO history_data (device_id, register_name, value, timestamp)
                    VALUES (?, ?, ?, ?)
                ''', (device_id, register_name, value, timestamp))

    def get_latest_data(self, device_id: str) -> dict[str, Any]:
        """获取设备最新数据"""
        with self.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('''
                SELECT register_name, value, unit, timestamp
                FROM realtime_data
                WHERE device_id = ?
            ''', (device_id,))

            result = {}
            for row in cursor.fetchall():
                result[row['register_name']] = {
                    'value': row['value'],
                    'unit': row['unit'],
                    'timestamp': row['timestamp']
                }
            return result

    def get_history_data(self, device_id: str, register_name: str = None,
                         start_time: datetime = None, end_time: datetime = None,
                         limit: int = 1000) -> list[dict[str, Any]]:
        """
        获取历史数据

        Args:
            device_id: 设备ID
            register_name: 寄存器名称（可选）
            start_time: 开始时间（可选）
            end_time: 结束时间（可选）
            limit: 返回条数限制

        Returns:
            历史数据列表
        """
        with self.get_connection() as conn:
            cursor = conn.cursor()

            # 构建查询条件
            conditions = ['device_id = ?']
            params = [device_id]

            if register_name:
                conditions.append('register_name = ?')
                params.append(register_name)

            if start_time:
                conditions.append('timestamp >= ?')
                params.append(start_time.isoformat())

            if end_time:
                conditions.append('timestamp <= ?')
                params.append(end_time.isoformat())

            where_clause = ' AND '.join(conditions)

            cursor.execute(f'''
                SELECT register_name, value, unit, timestamp
                FROM history_data
                WHERE {where_clause}
                ORDER BY timestamp DESC
                LIMIT ?
            ''', params + [limit])

            return [dict(row) for row in cursor.fetchall()]

    def get_alarm_records(self, start_time: datetime = None, end_time: datetime = None,
                          device_id: str = None, limit: int = 1000) -> list[dict[str, Any]]:
        """获取报警记录"""
        with self.get_connection() as conn:
            cursor = conn.cursor()

            conditions = []
            params = []

            if start_time:
                conditions.append('timestamp >= ?')
                params.append(start_time.isoformat())

            if end_time:
                conditions.append('timestamp <= ?')
                params.append(end_time.isoformat())

            if device_id:
                conditions.append('device_id = ?')
                params.append(device_id)

            where_clause = ' AND '.join(conditions) if conditions else '1=1'

            cursor.execute(f'''
                SELECT * FROM alarm_records
                WHERE {where_clause}
                ORDER BY timestamp DESC
                LIMIT ?
            ''', params + [limit])

            return [dict(row) for row in cursor.fetchall()]

    def save_alarm(self, alarm_data: dict[str, Any]):
        """保存报警记录"""
        with self.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('''
                INSERT INTO alarm_records 
                (alarm_id, device_id, register_name, alarm_level, 
                 alarm_message, threshold, actual_value, timestamp)
                VALUES (?, ?, ?, ?, ?, ?, ?, ?)
            ''', (
                alarm_data['alarm_id'],
                alarm_data['device_id'],
                alarm_data['register_name'],
                alarm_data['level'],
                alarm_data['message'],
                alarm_data.get('threshold'),
                alarm_data.get('actual_value'),
                alarm_data['timestamp']
            ))

    def cleanup_old_data(self, retention_days: int = 30):
        """清理过期数据"""
        cutoff_date = datetime.now() - timedelta(days=retention_days)

        with self.get_connection() as conn:
            cursor = conn.cursor()

            # 清理历史数据
            cursor.execute('''
                DELETE FROM history_data WHERE timestamp < ?
            ''', (cutoff_date.isoformat(),))
            deleted_count = cursor.rowcount

            logger.info(f"已清理 {deleted_count} 条过期历史数据")
            return deleted_count

    def get_database_stats(self) -> dict[str, Any]:
        """获取数据库统计信息"""
        with self.get_connection() as conn:
            cursor = conn.cursor()

            stats = {}

            # 各表记录数
            for table in ['realtime_data', 'history_data', 'alarm_records', 'device_status']:
                cursor.execute(f'SELECT COUNT(*) FROM {table}')
                stats[f'{table}_count'] = cursor.fetchone()[0]

            # 数据库文件大小
            stats['db_file_size'] = Path(self.db_path).stat().st_size

            return stats
```

第12章：数据归档模块
---------------

### 12.1 模块概述

数据归档模块负责将历史数据进行压缩和归档，节省存储空间。

**核心功能**：
1. **数据压缩**：使用多种算法压缩数据（滑动平均、最大值保留、LTTB等）
2. **数据归档**：将旧数据移动到归档表，释放主表空间
3. **数据清理**：删除过期数据

**生活比喻**：
数据归档就像"整理旧文件"，把不常用的文件（旧数据）压缩打包（归档），放到"储藏室"（归档表），腾出"办公桌"（主表）的空间。

### 12.2 完整源代码

```python
"""
数据归档模块
实现数据压缩和归档功能
支持多种压缩算法：滑动平均、最大值保留、最小值保留、LTTB降采样
"""

import logging
import math
from datetime import datetime, timedelta
from typing import Any, List, Dict, Tuple
from collections import defaultdict

logger = logging.getLogger(__name__)


class DataArchive:
    """
    数据归档类
    负责数据压缩和归档
    
    压缩算法：
    1. 滑动平均 (moving_average) - 适用于平稳数据
    2. 最大值保留 (max_keep) - 适用于需要监控峰值的场景
    3. 最小值保留 (min_keep) - 适用于需要监控谷值的场景
    4. LTTB (Largest Triangle Three Buckets) - 适用于趋势图显示
    5. 统计聚合 (statistical) - 保留均值、最大、最小、标准差
    """

    def __init__(self, database):
        """
        初始化数据归档

        Args:
            database: 数据库实例
        """
        self.database = database
        
        # 压缩算法映射
        self.compress_algorithms = {
            'moving_average': self._compress_moving_average,
            'max_keep': self._compress_max_keep,
            'min_keep': self._compress_min_keep,
            'lttb': self._compress_lttb,
            'statistical': self._compress_statistical,
        }

    def archive_data(self, retention_days: int = 30, archive_table: str = 'history_data_archive'):
        """
        归档旧数据
        
        将超过保留天数的数据移动到归档表，然后删除原表数据

        Args:
            retention_days: 数据保留天数
            archive_table: 归档表名
        """
        logger.info(f"开始归档 {retention_days} 天前的数据...")
        
        cutoff_date = datetime.now() - timedelta(days=retention_days)
        
        # 创建归档表（如果不存在）
        self._create_archive_table(archive_table)
        
        # 移动数据到归档表
        moved_count = self._move_to_archive(cutoff_date, archive_table)
        
        # 删除原表中的旧数据
        deleted_count = self.database.cleanup_old_data(retention_days)
        
        logger.info(f"数据归档完成: 移动 {moved_count} 条到归档表，删除 {deleted_count} 条旧数据")
        
        return {
            'moved_to_archive': moved_count,
            'deleted_from_main': deleted_count,
            'cutoff_date': cutoff_date.isoformat()
        }

    def _create_archive_table(self, archive_table: str):
        """创建归档表"""
        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute(f'''
                CREATE TABLE IF NOT EXISTS {archive_table} (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    device_id TEXT NOT NULL,
                    register_name TEXT NOT NULL,
                    value REAL,
                    unit TEXT,
                    timestamp DATETIME NOT NULL,
                    archived_at DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')
            
            # 创建索引
            cursor.execute(f'''
                CREATE INDEX IF NOT EXISTS idx_{archive_table}_device_time 
                ON {archive_table}(device_id, register_name, timestamp)
            ''')

    def _move_to_archive(self, cutoff_date: datetime, archive_table: str) -> int:
        """移动数据到归档表"""
        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            
            # 插入到归档表
            cursor.execute(f'''
                INSERT INTO {archive_table} (device_id, register_name, value, unit, timestamp)
                SELECT device_id, register_name, value, unit, timestamp
                FROM history_data
                WHERE timestamp < ?
            ''', (cutoff_date.isoformat(),))
            
            moved_count = cursor.rowcount
            
            return moved_count

    def compress_data(self, device_id: str, register_name: str,
                      start_time: datetime, end_time: datetime,
                      interval: str = '1hour', algorithm: str = 'statistical') -> Dict[str, Any]:
        """
        压缩数据

        Args:
            device_id: 设备ID
            register_name: 寄存器名称
            start_time: 开始时间
            end_time: 结束时间
            interval: 压缩间隔 (1min, 5min, 15min, 1hour, 1day)
            algorithm: 压缩算法 (moving_average, max_keep, min_keep, lttb, statistical)

        Returns:
            压缩结果
        """
        # 获取原始数据
        raw_data = self.database.get_history_data(
            device_id=device_id,
            register_name=register_name,
            start_time=start_time,
            end_time=end_time,
            interval='1min'  # 获取1分钟粒度的原始数据
        )
        
        if not raw_data:
            return {
                'device_id': device_id,
                'register_name': register_name,
                'original_count': 0,
                'compressed_count': 0,
                'compression_ratio': 0,
                'data': []
            }
        
        # 执行压缩
        compress_func = self.compress_algorithms.get(algorithm)
        if compress_func is None:
            logger.error(f"不支持的压缩算法: {algorithm}")
            return None
        
        compressed_data = compress_func(raw_data, interval)
        
        # 计算压缩比
        original_count = len(raw_data)
        compressed_count = len(compressed_data)
        compression_ratio = compressed_count / original_count if original_count > 0 else 0
        
        return {
            'device_id': device_id,
            'register_name': register_name,
            'original_count': original_count,
            'compressed_count': compressed_count,
            'compression_ratio': round(compression_ratio, 4),
            'algorithm': algorithm,
            'interval': interval,
            'data': compressed_data
        }

    def _compress_moving_average(self, data: List[Dict], interval: str) -> List[Dict]:
        """
        滑动平均压缩

        将多个数据点合并为一个，取平均值。
        适用于平稳数据，可以有效去除噪声。
        """
        interval_seconds = self._parse_interval(interval)
        if not data:
            return []
        
        result = []
        current_bucket = []
        bucket_start = data[0]['timestamp']
        
        for item in data:
            item_time = item['timestamp']
            if isinstance(item_time, str):
                item_time = datetime.fromisoformat(item_time)
            
            # 检查是否需要开始新的桶
            if (item_time - bucket_start).total_seconds() >= interval_seconds:
                if current_bucket:
                    # 计算平均值
                    avg_value = sum(d['value'] for d in current_bucket) / len(current_bucket)
                    result.append({
                        'timestamp': bucket_start.isoformat(),
                        'value': round(avg_value, 4),
                        'count': len(current_bucket)
                    })
                bucket_start = item_time
                current_bucket = []
            
            current_bucket.append(item)
        
        # 处理最后一个桶
        if current_bucket:
            avg_value = sum(d['value'] for d in current_bucket) / len(current_bucket)
            result.append({
                'timestamp': bucket_start.isoformat(),
                'value': round(avg_value, 4),
                'count': len(current_bucket)
            })
        
        return result

    def _compress_max_keep(self, data: List[Dict], interval: str) -> List[Dict]:
        """
        最大值保留压缩

        在每个时间桶内保留最大值。
        适用于需要监控峰值的场景（如温度监控）。
        """
        interval_seconds = self._parse_interval(interval)
        if not data:
            return []
        
        result = []
        current_bucket = []
        bucket_start = data[0]['timestamp']
        
        for item in data:
            item_time = item['timestamp']
            if isinstance(item_time, str):
                item_time = datetime.fromisoformat(item_time)
            
            if (item_time - bucket_start).total_seconds() >= interval_seconds:
                if current_bucket:
                    max_item = max(current_bucket, key=lambda x: x['value'])
                    result.append({
                        'timestamp': max_item['timestamp'],
                        'value': max_item['value'],
                        'count': len(current_bucket)
                    })
                bucket_start = item_time
                current_bucket = []
            
            current_bucket.append(item)
        
        if current_bucket:
            max_item = max(current_bucket, key=lambda x: x['value'])
            result.append({
                'timestamp': max_item['timestamp'],
                'value': max_item['value'],
                'count': len(current_bucket)
            })
        
        return result

    def _compress_min_keep(self, data: List[Dict], interval: str) -> List[Dict]:
        """
        最小值保留压缩

        在每个时间桶内保留最小值。
        适用于需要监控谷值的场景（如压力监控）。
        """
        interval_seconds = self._parse_interval(interval)
        if not data:
            return []
        
        result = []
        current_bucket = []
        bucket_start = data[0]['timestamp']
        
        for item in data:
            item_time = item['timestamp']
            if isinstance(item_time, str):
                item_time = datetime.fromisoformat(item_time)
            
            if (item_time - bucket_start).total_seconds() >= interval_seconds:
                if current_bucket:
                    min_item = min(current_bucket, key=lambda x: x['value'])
                    result.append({
                        'timestamp': min_item['timestamp'],
                        'value': min_item['value'],
                        'count': len(current_bucket)
                    })
                bucket_start = item_time
                current_bucket = []
            
            current_bucket.append(item)
        
        if current_bucket:
            min_item = min(current_bucket, key=lambda x: x['value'])
            result.append({
                'timestamp': min_item['timestamp'],
                'value': min_item['value'],
                'count': len(current_bucket)
            })
        
        return result

    def _compress_lttb(self, data: List[Dict], interval: str) -> List[Dict]:
        """
        LTTB (Largest Triangle Three Buckets) 压缩算法

        这是一种智能降采样算法，能够保留数据的视觉特征。
        适用于趋势图显示，能够在大幅减少数据点的同时，
        保持曲线的整体形状和重要特征。
        """
        interval_seconds = self._parse_interval(interval)
        if len(data) <= 2:
            return data
        
        result = [data[0]]  # 保留第一个点
        
        bucket_size = max(1, int(interval_seconds / 60))  # 假设原始数据是1分钟粒度
        
        for i in range(1, len(data) - 1, bucket_size):
            bucket = data[i:i + bucket_size]
            if not bucket:
                continue
            
            # 选择与前后点形成最大三角形面积的点
            prev_point = result[-1]
            next_bucket = data[i + bucket_size:i + 2 * bucket_size]
            
            if not next_bucket:
                next_point = data[-1]
            else:
                next_point = next_bucket[len(next_bucket) // 2]
            
            max_area = -1
            selected_point = bucket[0]
            
            for point in bucket:
                area = abs(
                    (prev_point['timestamp'].timestamp() - next_point['timestamp'].timestamp()) *
                    (point['value'] - prev_point['value']) -
                    (prev_point['timestamp'].timestamp() - point['timestamp'].timestamp()) *
                    (next_point['value'] - prev_point['value'])
                )
                if area > max_area:
                    max_area = area
                    selected_point = point
            
            result.append(selected_point)
        
        result.append(data[-1])  # 保留最后一个点
        
        return result

    def _compress_statistical(self, data: List[Dict], interval: str) -> List[Dict]:
        """
        统计聚合压缩

        在每个时间桶内计算统计值：均值、最大值、最小值、标准差。
        适用于需要保留数据分布特征的场景。
        """
        interval_seconds = self._parse_interval(interval)
        if not data:
            return []
        
        result = []
        current_bucket = []
        bucket_start = data[0]['timestamp']
        
        for item in data:
            item_time = item['timestamp']
            if isinstance(item_time, str):
                item_time = datetime.fromisoformat(item_time)
            
            if (item_time - bucket_start).total_seconds() >= interval_seconds:
                if current_bucket:
                    values = [d['value'] for d in current_bucket]
                    avg = sum(values) / len(values)
                    std = math.sqrt(sum((x - avg) ** 2 for x in values) / len(values))
                    
                    result.append({
                        'timestamp': bucket_start.isoformat(),
                        'mean': round(avg, 4),
                        'max': max(values),
                        'min': min(values),
                        'std': round(std, 4),
                        'count': len(current_bucket)
                    })
                bucket_start = item_time
                current_bucket = []
            
            current_bucket.append(item)
        
        if current_bucket:
            values = [d['value'] for d in current_bucket]
            avg = sum(values) / len(values)
            std = math.sqrt(sum((x - avg) ** 2 for x in values) / len(values))
            
            result.append({
                'timestamp': bucket_start.isoformat(),
                'mean': round(avg, 4),
                'max': max(values),
                'min': min(values),
                'std': round(std, 4),
                'count': len(current_bucket)
            })
        
        return result

    def _parse_interval(self, interval: str) -> int:
        """解析时间间隔字符串为秒数"""
        interval_map = {
            '1min': 60,
            '5min': 300,
            '15min': 900,
            '1hour': 3600,
            '1day': 86400
        }
        return interval_map.get(interval, 3600)
```

### 12.3 代码详解

**压缩算法对比**：

| 算法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 滑动平均 | 平稳数据 | 去除噪声 | 可能丢失峰值 |
| 最大值保留 | 温度监控 | 保留峰值 | 数据量减少有限 |
| 最小值保留 | 压力监控 | 保留谷值 | 数据量减少有限 |
| LTTB | 趋势图显示 | 保留视觉特征 | 算法复杂 |
| 统计聚合 | 数据分析 | 保留分布特征 | 数据量较大 |

### 14.1 模块概述

设备控制模块是SCADA系统的"安全卫士"，负责设备的安全控制和联锁保护。

**核心功能**：
1. **安全联锁**：当参数超限时，自动执行保护动作（如关闭加热器、停止泵）
2. **写操作安全校验**：检查写入值是否在安全范围内
3. **紧急停机（E-STOP）**：一键停机，独立通道，最高优先级
4. **操作审计**：记录所有操作，便于追溯

**生活比喻**：
安全联锁就像家里的"烟雾报警器"，当检测到烟雾（温度超限）时，自动切断电源（关闭加热器），防止火灾发生。

### 14.2 核心代码片段

```python
class DeviceControlSafety:
    """
    工厂级设备控制安全管理器

    职责：
    - 安全联锁规则管理与执行
    - 写操作安全校验
    - 故障降级策略
    - 紧急停机管理
    - 操作审计日志
    """

    def __init__(self, database, device_manager=None, alarm_manager=None):
        self.database = database
        self.device_manager = device_manager
        self.alarm_manager = alarm_manager

        # 安全联锁规则
        self._interlock_rules: dict[str, dict[str, Any]] = {}
        self._interlock_states: dict[str, bool] = {}  # rule_id -> is_triggered

        # 写操作安全配置
        self._write_limits: dict[str, dict[str, Any]] = {}
        self._write_whitelist: dict[str, Set[int]] = {}

        # 操作互斥锁
        self._device_locks: dict[str, threading.Lock] = {}

        # 紧急停机
        self._estop_active = False
        self._estop_time: datetime | None = None

        # 操作审计
        self._audit_log = deque(maxlen=5000)

        # 加载预置联锁规则
        self._load_preset_interlocks()

    def _load_preset_interlocks(self):
        """加载预置安全联锁规则（工厂标准配置）"""
        preset_rules = [
            {
                'id': 'interlock_temp_high',
                'name': '温度超限联锁',
                'description': '反应釜温度超过上限自动停加热',
                'priority': 1,
                'condition': {
                    'type': 'threshold',
                    'device_id': 'plc_reactor_01',
                    'register': 'temperature',
                    'operator': '>',
                    'value': 95.0
                },
                'action': {
                    'type': 'write_register',
                    'device_id': 'plc_reactor_01',
                    'register': 'heater_enable',
                    'value': 0
                },
                'alarm': {
                    'level': 'critical',
                    'message': '温度超限联锁触发：反应釜温度>{threshold}°C，加热已自动关闭'
                }
            },
            {
                'id': 'interlock_pressure_high',
                'name': '压力超限联锁',
                'description': '管道压力超过上限自动停泵',
                'priority': 1,
                'condition': {
                    'type': 'threshold',
                    'device_id': 'plc_reactor_01',
                    'register': 'pressure',
                    'operator': '>',
                    'value': 2.5
                },
                'action': {
                    'type': 'write_register',
                    'device_id': 'plc_reactor_01',
                    'register': 'pump_enable',
                    'value': 0
                },
                'alarm': {
                    'level': 'critical',
                    'message': '压力超限联锁触发：管道压力>{threshold}MPa，泵已自动停止'
                }
            }
        ]

        for rule in preset_rules:
            self._interlock_rules[rule['id']] = rule
            self._interlock_states[rule['id']] = False

        logger.info(f"已加载 {len(preset_rules)} 条预置联锁规则")

    def check_interlocks(self, device_id: str, register_name: str, value: float):
        """
        检查联锁条件

        当设备数据更新时调用，检查是否触发联锁。
        如果触发，自动执行保护动作。

        Args:
            device_id: 设备ID
            register_name: 寄存器名称
            value: 当前值
        """
        for rule_id, rule in self._interlock_rules.items():
            if not rule.get('enabled', True):
                continue

            condition = rule['condition']
            if (condition['device_id'] == device_id and
                condition['register'] == register_name):

                # 检查条件
                triggered = self._evaluate_condition(condition, value)

                if triggered and not self._interlock_states[rule_id]:
                    # 联锁触发
                    self._interlock_states[rule_id] = True
                    self._execute_interlock_action(rule, value)
                    logger.warning(f"联锁触发: {rule['name']} - {register_name}={value}")

                elif not triggered and self._interlock_states[rule_id]:
                    # 联锁恢复
                    self._interlock_states[rule_id] = False
                    logger.info(f"联锁恢复: {rule['name']}")

    def _evaluate_condition(self, condition: dict, value: float) -> bool:
        """评估联锁条件"""
        operator = condition['operator']
        threshold = condition['value']

        if operator == '>':
            return value > threshold
        elif operator == '<':
            return value < threshold
        elif operator == '>=':
            return value >= threshold
        elif operator == '<=':
            return value <= threshold
        elif operator == '==':
            return abs(value - threshold) < 0.001
        return False

    def _execute_interlock_action(self, rule: dict, actual_value: float):
        """执行联锁动作"""
        action = rule['action']

        if action['type'] == 'write_register':
            # 写寄存器（如关闭加热器、停止泵）
            device_id = action['device_id']
            register = action['register']
            value = action['value']

            client = self.device_manager.get_client(device_id)
            if client:
                client.write_register(register, value)
                logger.info(f"联锁动作: {device_id}.{register} = {value}")

        # 发送报警
        if 'alarm' in rule:
            alarm_info = rule['alarm']
            message = alarm_info['message'].format(
                threshold=rule['condition']['value'],
                actual=actual_value
            )
            if self.alarm_manager:
                self.alarm_manager.trigger_alarm(
                    device_id=rule['condition']['device_id'],
                    register_name=rule['condition']['register'],
                    level=alarm_info['level'],
                    message=message,
                    actual_value=actual_value,
                    threshold=rule['condition']['value']
                )

    def trigger_estop(self, reason: str = ''):
        """
        触发紧急停机

        E-STOP是最高优先级的安全功能：
        1. 立即停止所有设备
        2. 独立通道，不依赖正常控制链路
        3. 需要手动复位才能恢复
        """
        self._estop_active = True
        self._estop_time = datetime.now()
        self._estop_reason = reason

        logger.critical(f"紧急停机触发: {reason}")

        # 停止所有设备
        for device_id in self.device_manager.get_all_devices():
            client = self.device_manager.get_client(device_id)
            if client:
                try:
                    client.write_register('estop', 1)
                except Exception as e:
                    logger.error(f"设备 {device_id} 紧急停机失败: {e}")

        # 发送紧急报警
        if self.alarm_manager:
            self.alarm_manager.trigger_alarm(
                device_id='SYSTEM',
                register_name='estop',
                level='emergency',
                message=f'紧急停机: {reason}'
            )

    def reset_estop(self):
        """重置紧急停机"""
        if not self._estop_active:
            return

        self._estop_active = False
        self._estop_time = None
        self._estop_reason = ''

        logger.info("紧急停机已重置")
```

第15章：OEE计算器
--------------

### 15.1 模块概述

OEE（Overall Equipment Effectiveness）是工业4.0的核心指标，衡量设备的综合效率。

**OEE公式**：
```
OEE = 可用率(A) × 性能率(P) × 质量率(Q)
```

**世界级OEE标准**：≥85%
- 可用率 ≥ 90%
- 性能率 ≥ 95%
- 质量率 ≥ 99.9%

### 15.2 核心代码片段

```python
class OEECalculator:
    """
    OEE设备综合效率计算器

    数据来源：
    - 设备运行状态（运行/停机/故障/待料）
    - 实际产量 vs 理论产量
    - 合格品数 vs 总产量
    """

    def __init__(self, database, config=None):
        self.database = database
        self.config = config or {}

        # 设备理论产能配置
        self.theoretical_rates = self.config.get('theoretical_rates', {})

        # 设备运行状态追踪
        self.device_states = {}

        # 班次数据累积
        self.shift_data = defaultdict(lambda: {
            'shift_start': None,
            'planned_production_time': 0,
            'actual_run_time': 0,
            'downtime': 0,
            'total_count': 0,
            'good_count': 0,
            'ideal_cycle_time': 60.0,
        })

    def calculate_oee(self, device_id: str) -> dict[str, Any] | None:
        """
        计算设备OEE

        Returns:
            {
                'device_id': str,
                'availability': float,  # 可用率
                'performance': float,   # 性能率
                'quality': float,       # 质量率
                'oee': float,           # 综合效率
                'losses': dict          # 六大损失
            }
        """
        sd = self.shift_data[device_id]
        if sd['planned_production_time'] <= 0:
            return None

        # 1. 可用率 = 实际运行时间 / 计划生产时间
        availability = sd['actual_run_time'] / sd['planned_production_time']

        # 2. 性能率 = (理想节拍 × 总产量) / 实际运行时间
        if sd['actual_run_time'] > 0 and sd['total_count'] > 0:
            performance = (sd['ideal_cycle_time'] * sd['total_count']) / sd['actual_run_time']
            performance = min(performance, 1.0)  # 性能率不超过100%
        else:
            performance = 0

        # 3. 质量率 = 合格品数 / 总产量
        if sd['total_count'] > 0:
            quality = sd['good_count'] / sd['total_count']
        else:
            quality = 0

        # 4. OEE = A × P × Q
        oee = availability * performance * quality

        # 5. 计算六大损失
        losses = {
            'equipment_failure': sd['downtime'],  # 设备故障
            'setup_adjustment': 0,                 # 换装调整
            'idling_minor_stop': 0,                # 空转小停
            'reduced_speed': sd['actual_run_time'] * (1 - performance),  # 速度降低
            'process_defects': sd['total_count'] - sd['good_count'],     # 过程缺陷
            'startup_rejects': 0,                  # 启动废品
        }

        return {
            'device_id': device_id,
            'availability': round(availability * 100, 2),
            'performance': round(performance * 100, 2),
            'quality': round(quality * 100, 2),
            'oee': round(oee * 100, 2),
            'losses': losses,
            'shift_data': dict(sd)
        }
```

### 16.1 模块概述

SPC（Statistical Process Control）是工业4.0的质量管理核心，用统计方法监控和控制生产过程。

**核心功能**：
1. **控制图**：X̄-R图、X̄-S图、p图、c图
2. **过程能力分析**：Cp、Cpk、Pp、Ppk
3. **Western Electric判异规则**：8大判异准则

### 16.2 核心代码片段

```python
class SPCAnalyzer:
    """
    SPC统计过程控制分析器

    工作流程：
    1. 持续采集质量数据
    2. 计算控制限（UCL/CL/LCL）
    3. 判异检测（Western Electric规则）
    4. 计算过程能力指数
    """

    def __init__(self, database, config=None):
        self.database = database
        self.config = config or {}

        # 控制图数据窗口
        self.window_size = self.config.get('window_size', 25)
        self.subgroup_size = self.config.get('subgroup_size', 5)

        # 数据缓冲区
        self.data_buffers = defaultdict(
            lambda: deque(maxlen=self.window_size * self.subgroup_size)
        )

        # 规格限配置（USL/LSL）
        self.spec_limits = self.config.get('spec_limits', {})

    def calculate_xbar_r_chart(self, device_id, register_name):
        """
        X̄-R控制图计算

        适用于子组大小 n=2~10
        """
        key = f"{device_id}:{register_name}"
        raw_data = list(self.data_buffers.get(key, []))

        if len(raw_data) < self.subgroup_size * 2:
            return None

        # 分组
        subgroups = self._form_subgroups(raw_data)

        # 计算每个子组的均值和极差
        xbar_values = []
        r_values = []
        for group in subgroups:
            xbar = sum(group) / len(group)
            r = max(group) - min(group)
            xbar_values.append(xbar)
            r_values.append(r)

        # 计算控制限
        xbar_bar = sum(xbar_values) / len(xbar_values)
        r_bar = sum(r_values) / len(r_values)

        # A2, D3, D4 系数（n=5时）
        A2 = 0.577
        D3 = 0
        D4 = 2.114

        xbar_ucl = xbar_bar + A2 * r_bar
        xbar_lcl = xbar_bar - A2 * r_bar
        r_ucl = D4 * r_bar
        r_lcl = D3 * r_bar

        return {
            'xbar': {
                'ucl': round(xbar_ucl, 4),
                'cl': round(xbar_bar, 4),
                'lcl': round(xbar_lcl, 4),
                'points': [round(v, 4) for v in xbar_values]
            },
            'r_chart': {
                'ucl': round(r_ucl, 4),
                'cl': round(r_bar, 4),
                'lcl': round(r_lcl, 4),
                'points': [round(v, 4) for v in r_values]
            }
        }

    def calculate_capability(self, device_id, register_name):
        """
        计算过程能力指数

        Cp = (USL - LSL) / (6σ)
        Cpk = min((USL - μ) / 3σ, (μ - LSL) / 3σ)
        """
        key = f"{device_id}:{register_name}"
        spec = self.spec_limits.get(key)
        if not spec:
            return None

        usl = spec.get('usl')
        lsl = spec.get('lsl')
        if not usl or not lsl:
            return None

        raw_data = list(self.data_buffers.get(key, []))
        if len(raw_data) < 30:
            return None

        mu = sum(raw_data) / len(raw_data)
        sigma = (sum((x - mu) ** 2 for x in raw_data) / len(raw_data)) ** 0.5

        if sigma == 0:
            return None

        cp = (usl - lsl) / (6 * sigma)
        cpk = min((usl - mu) / (3 * sigma), (mu - lsl) / (3 * sigma))

        return {
            'cp': round(cp, 4),
            'cpk': round(cpk, 4),
            'usl': usl,
            'lsl': lsl,
            'mean': round(mu, 4),
            'sigma': round(sigma, 4),
            'sample_size': len(raw_data)
        }
```

第17章：预测性维护
--------------

### 17.1 模块概述

预测性维护是工业4.0的核心能力，基于数据驱动预测设备何时需要维护。

**核心功能**：
1. **趋势分析**：移动平均 + 线性回归预测未来值
2. **异常检测**：Z-Score + IQR双算法
3. **设备健康评分**：多维度加权评估 (0-100)
4. **故障预测**：基于趋势外推预测何时超限

### 17.2 核心代码片段

```python
class PredictiveMaintenance:
    """
    预测性维护引擎

    工作流程：
    1. 持续采集设备数据 → 存入滑动窗口
    2. 定期执行趋势分析 + 异常检测
    3. 计算设备健康评分
    4. 预测故障时间窗口
    5. 生成维护建议
    """

    def __init__(self, database, config=None):
        self.database = database
        self.config = config or {}

        # 滑动窗口
        self.window_size = self.config.get('window_size', 500)
        self.data_windows = defaultdict(
            lambda: deque(maxlen=self.window_size)
        )

        # 设备健康评分缓存
        self.health_scores = {}

        # 维护建议队列
        self.maintenance_alerts = []

    def feed_data(self, device_id, register_name, value, timestamp=None):
        """喂入实时数据"""
        key = f"{device_id}:{register_name}"
        ts = timestamp or datetime.now()
        self.data_windows[key].append({
            'value': value,
            'timestamp': ts,
            'ts_epoch': ts.timestamp()
        })

    def _analyze_trend(self, window):
        """
        趋势分析：使用线性回归预测未来值

        返回：
        - slope: 斜率（正值上升，负值下降）
        - intercept: 截距
        - r_squared: R²值（拟合优度）
        - predicted_value: 预测的下一个值
        """
        n = len(window)
        if n < 10:
            return None

        values = [d['value'] for d in window]
        timestamps = [d['ts_epoch'] for d in window]

        # 简单线性回归
        x_mean = sum(timestamps) / n
        y_mean = sum(values) / n

        numerator = sum((x - x_mean) * (y - y_mean) for x, y in zip(timestamps, values))
        denominator = sum((x - x_mean) ** 2 for x in timestamps)

        if denominator == 0:
            return None

        slope = numerator / denominator
        intercept = y_mean - slope * x_mean

        # 计算R²
        ss_res = sum((y - (slope * x + intercept)) ** 2 for x, y in zip(timestamps, values))
        ss_tot = sum((y - y_mean) ** 2 for y in values)
        r_squared = 1 - (ss_res / ss_tot) if ss_tot > 0 else 0

        # 预测下一个值
        next_ts = timestamps[-1] + (timestamps[-1] - timestamps[-2])
        predicted = slope * next_ts + intercept

        return {
            'slope': slope,
            'intercept': intercept,
            'r_squared': r_squared,
            'predicted_value': predicted,
            'trend': 'rising' if slope > 0 else 'falling' if slope < 0 else 'stable'
        }

    def _detect_anomalies(self, window):
        """
        异常检测：Z-Score + IQR双算法

        Z-Score: |x - μ| / σ > 3 为异常
        IQR: x < Q1 - 1.5*IQR 或 x > Q3 + 1.5*IQR 为异常
        """
        values = [d['value'] for d in window]
        n = len(values)

        if n < 10:
            return []

        # Z-Score检测
        mean = sum(values) / n
        std = (sum((x - mean) ** 2 for x in values) / n) ** 0.5
        z_anomalies = []
        if std > 0:
            for i, v in enumerate(values):
                z_score = abs(v - mean) / std
                if z_score > 3:
                    z_anomalies.append({
                        'index': i,
                        'value': v,
                        'z_score': round(z_score, 2),
                        'method': 'z-score'
                    })

        # IQR检测
        sorted_values = sorted(values)
        q1 = sorted_values[n // 4]
        q3 = sorted_values[3 * n // 4]
        iqr = q3 - q1
        lower_bound = q1 - 1.5 * iqr
        upper_bound = q3 + 1.5 * iqr

        iqr_anomalies = []
        for i, v in enumerate(values):
            if v < lower_bound or v > upper_bound:
                iqr_anomalies.append({
                    'index': i,
                    'value': v,
                    'bounds': (round(lower_bound, 2), round(upper_bound, 2)),
                    'method': 'iqr'
                })

        return z_anomalies + iqr_anomalies

    def calculate_health_score(self, device_id):
        """
        计算设备健康评分（0-100）

        评分维度：
        1. 趋势稳定性（30%）
        2. 异常频率（30%）
        3. 参数范围（20%）
        4. 数据质量（20%）
        """
        scores = []
        weights = []

        for key, window in self.data_windows.items():
            if not key.startswith(f"{device_id}:"):
                continue

            if len(window) < 10:
                continue

            values = [d['value'] for d in window]

            # 1. 趋势稳定性（斜率越小越好）
            trend = self._analyze_trend(list(window))
            if trend:
                trend_score = max(0, 100 - abs(trend['slope']) * 1000)
                scores.append(trend_score)
                weights.append(0.3)

            # 2. 异常频率（异常越少越好）
            anomalies = self._detect_anomalies(list(window))
            anomaly_rate = len(anomalies) / len(values)
            anomaly_score = max(0, 100 - anomaly_rate * 1000)
            scores.append(anomaly_score)
            weights.append(0.3)

            # 3. 参数范围（在正常范围内得分高）
            mean = sum(values) / len(values)
            std = (sum((x - mean) ** 2 for x in values) / len(values)) ** 0.5
            range_score = max(0, 100 - std * 10)
            scores.append(range_score)
            weights.append(0.2)

            # 4. 数据质量（数据点越多越好）
            quality_score = min(100, len(values) / self.window_size * 100)
            scores.append(quality_score)
            weights.append(0.2)

        if not scores:
            return None

        # 加权平均
        total_score = sum(s * w for s, w in zip(scores, weights)) / sum(weights)

        return {
            'device_id': device_id,
            'health_score': round(total_score, 1),
            'status': 'healthy' if total_score >= 80 else 'warning' if total_score >= 60 else 'critical',
            'details': {
                'trend_stability': round(scores[0], 1) if len(scores) > 0 else None,
                'anomaly_frequency': round(scores[1], 1) if len(scores) > 1 else None,
                'parameter_range': round(scores[2], 1) if len(scores) > 2 else None,
                'data_quality': round(scores[3], 1) if len(scores) > 3 else None
            }
        }
```

```python
class EnergyManager:
    """
    能源管理引擎

    数据来源：
    - 电力仪表（功率、电量、功率因数）
    - 水表/气表/蒸汽表（流量、累积量）
    - 设备运行状态（关联能耗与产出）
    """

    # 默认电价配置
    DEFAULT_TARIFF = {
        'peak': 1.2,    # 峰时电价（元/kWh）
        'flat': 0.7,    # 平时电价
        'valley': 0.35  # 谷时电价
    }

    # 峰谷平时段定义
    DEFAULT_PERIODS = {
        'peak': [(8, 11), (18, 23)],
        'valley': [(0, 7), (23, 24)],
    }

    def __init__(self, database, config=None, config_path='配置/energy.yaml'):
        self.database = database
        self.config_path = Path(config_path)

        # 加载配置
        self._load_config(config)

        # 实时功率数据
        self.realtime_power = {}

        # 能耗累积数据
        self.energy_accumulated = defaultdict(lambda: {
            'energy_kwh': 0, 'water_m3': 0, 'gas_m3': 0, 'steam_ton': 0,
            'peak_kwh': 0, 'flat_kwh': 0, 'valley_kwh': 0,
        })

    def feed_power_data(self, device_id, power_kw, timestamp=None):
        """
        喂入功率数据

        Args:
            device_id: 设备ID
            power_kw: 功率（kW）
            timestamp: 时间戳
        """
        now = timestamp or datetime.now()

        # 更新实时功率
        self.realtime_power[device_id] = {
            'power_kw': power_kw,
            'timestamp': now
        }

        # 累积电量（假设每秒采集一次）
        period_type = self._get_period_type(now)
        energy_kwh = power_kw / 3600  # 转换为kWh

        acc = self.energy_accumulated[device_id]
        acc['energy_kwh'] += energy_kwh
        acc[f'{period_type}_kwh'] += energy_kwh

    def _get_period_type(self, dt):
        """判断当前时间属于峰/谷/平"""
        hour = dt.hour

        for start, end in self.tariff_periods.get('peak', []):
            if start <= hour < end:
                return 'peak'

        for start, end in self.tariff_periods.get('valley', []):
            if start <= hour < end:
                return 'valley'

        return 'flat'

    def calculate_cost(self, device_id):
        """
        计算设备电费

        Returns:
            {
                'device_id': str,
                'total_cost': float,
                'peak_cost': float,
                'flat_cost': float,
                'valley_cost': float,
                'total_kwh': float
            }
        """
        acc = self.energy_accumulated[device_id]

        peak_cost = acc['peak_kwh'] * self.tariff['peak']
        flat_cost = acc['flat_kwh'] * self.tariff['flat']
        valley_cost = acc['valley_kwh'] * self.tariff['valley']
        total_cost = peak_cost + flat_cost + valley_cost

        return {
            'device_id': device_id,
            'total_cost': round(total_cost, 2),
            'peak_cost': round(peak_cost, 2),
            'flat_cost': round(flat_cost, 2),
            'valley_cost': round(valley_cost, 2),
            'total_kwh': round(acc['energy_kwh'], 2),
            'peak_kwh': round(acc['peak_kwh'], 2),
            'flat_kwh': round(acc['flat_kwh'], 2),
            'valley_kwh': round(acc['valley_kwh'], 2)
        }

    def calculate_carbon_emission(self, device_id):
        """
        计算碳排放

        碳排放 = 电量 × 碳排放因子
        默认碳排放因子：0.581 kgCO₂/kWh（国家电网平均值）
        """
        acc = self.energy_accumulated[device_id]
        emission = acc['energy_kwh'] * self.carbon_factor

        return {
            'device_id': device_id,
            'carbon_emission_kg': round(emission, 2),
            'carbon_factor': self.carbon_factor,
            'energy_kwh': round(acc['energy_kwh'], 2)
        }
```

第19章：边缘决策引擎
---------------

### 19.1 模块概述

边缘决策引擎是工业4.0去中心化决策的核心，实现本地自主决策，不依赖云端。

**决策层级**：
1. **安全联锁**（最高优先级，毫秒级响应）
2. **规则引擎**（秒级响应）
3. **自适应调节**（分钟级响应）

### 19.2 核心代码片段

```python
class EdgeDecisionEngine:
    """
    边缘决策引擎

    工业4.0要求：关键决策在边缘（本地）完成，不依赖云端
    """

    def __init__(self, database, config=None):
        self.database = database
        self.config = config or {}

        # 规则库
        self.rules = {}

        # 联锁规则库
        self.interlocks = {}

        # PID控制器库
        self.pid_controllers = {}

        # 决策日志
        self.decision_log = deque(maxlen=1000)

        # 当前数据快照
        self._data_snapshot = {}

    def add_rule(self, rule_id, condition, action, name='', priority=100, enabled=True):
        """
        添加决策规则

        Args:
            rule_id: 规则ID
            condition: 条件（如 {'type': 'threshold', 'key': 'device:temp', 'operator': 'gt', 'value': 80}）
            action: 动作（如 {'type': 'set_alarm', 'message': '温度过高'}）
            name: 规则名称
            priority: 优先级（数字越小优先级越高）
            enabled: 是否启用
        """
        self.rules[rule_id] = {
            'id': rule_id,
            'condition': condition,
            'action': action,
            'name': name,
            'priority': priority,
            'enabled': enabled,
            'last_triggered': None,
            'trigger_count': 0
        }

    def add_interlock(self, interlock_id, condition, action, name='', enabled=True):
        """添加安全联锁规则"""
        self.interlocks[interlock_id] = {
            'id': interlock_id,
            'condition': condition,
            'action': action,
            'name': name,
            'enabled': enabled,
            'is_triggered': False
        }

    def feed_data(self, device_id, data, timestamp=None):
        """喂入实时数据"""
        for register_name, value in data.items():
            if isinstance(value, (int, float)):
                key = f"{device_id}:{register_name}"
                self._data_snapshot[key] = value

    def _evaluate_condition(self, condition):
        """评估条件"""
        if condition['type'] == 'threshold':
            key = condition['key']
            value = self._data_snapshot.get(key)
            if value is None:
                return False

            operator = condition['operator']
            threshold = condition['value']

            if operator == 'gt':
                return value > threshold
            elif operator == 'lt':
                return value < threshold
            elif operator == 'gte':
                return value >= threshold
            elif operator == 'lte':
                return value <= threshold
            elif operator == 'eq':
                return abs(value - threshold) < 0.001

        return False

    def _execute_action(self, action, rule):
        """执行动作"""
        if action['type'] == 'set_alarm':
            logger.warning(f"边缘决策触发报警: {action['message']}")
            # 这里可以调用报警管理器

        elif action['type'] == 'write_register':
            logger.info(f"边缘决策写寄存器: {action['device']}.{action['register']} = {action['value']}")

    def _decision_loop(self):
        """决策主循环"""
        while self._running:
            try:
                # 1. 检查安全联锁（最高优先级）
                for interlock_id, interlock in self.interlocks.items():
                    if not interlock['enabled']:
                        continue

                    triggered = self._evaluate_condition(interlock['condition'])
                    if triggered and not interlock['is_triggered']:
                        interlock['is_triggered'] = True
                        self._execute_action(interlock['action'], interlock)
                        logger.critical(f"安全联锁触发: {interlock['name']}")

                # 2. 检查决策规则
                sorted_rules = sorted(self.rules.values(), key=lambda r: r['priority'])
                for rule in sorted_rules:
                    if not rule['enabled']:
                        continue

                    if self._evaluate_condition(rule['condition']):
                        self._execute_action(rule['action'], rule)
                        rule['trigger_count'] += 1
                        rule['last_triggered'] = datetime.now()

                time.sleep(1)  # 每秒检查一次

            except Exception as e:
                logger.error(f"决策循环异常: {e}")
                time.sleep(5)
```

第20章：报警管理器
--------------

### 20.1 模块概述

报警管理器是SCADA系统的"警报中心"，负责报警检测、记录、通知。

**核心改进**：
- **冷却窗口**：同一报警在冷却时间内只推送一次
- **确认抑制**：用户确认后，同一报警在抑制时间内不再弹窗
- **三级联动**：声光报警器 + 语音广播 + 前端推送

### 20.2 核心代码片段

```python
class AlarmDedupConfig:
    """报警去重配置"""
    def __init__(self, config=None):
        config = config or {}
        self.emit_cooldown_seconds = config.get('emit_cooldown_seconds', 60)
        self.acknowledge_suppress_seconds = config.get('acknowledge_suppress_seconds', 300)
        self.enabled = config.get('enabled', True)
        self.max_visible_toasts = config.get('max_visible_toasts', 3)


class AlarmManager:
    """
    报警管理器（重写版）
    负责报警检测、记录、声光输出、语音广播、前端推送

    报警输出总线（三级联动）：
    1. 声光报警器（Modbus DO -> 报警灯塔 + 蜂鸣器）
    2. 语音广播系统（MQTT -> IP网络广播/现场音柱）
    3. 前端WebSocket推送（页面弹窗/音效/语音合成）
    """

    def __init__(self, database, config_path='配置/alarms.yaml',
                 alarm_output=None, broadcast_system=None):
        self.database = database
        self.config_path = config_path

        # 报警规则
        self.rules = {}

        # 报警状态
        self.alarm_states = {}

        # 输出总线
        self.alarm_output = alarm_output
        self.broadcast_system = broadcast_system

        # 去重配置
        self.dedup_config = AlarmDedupConfig()

        # 去重状态
        self._emit_history = {}
        self._acknowledge_history = {}

        # 加载报警配置
        self.load_config()

    def check_alarms(self, device_id, data, timestamp):
        """
        检查报警

        遍历所有报警规则，检查设备数据是否触发报警。
        """
        for rule_id, rule in self.rules.items():
            if not rule.get('enabled', True):
                continue

            if rule['device_id'] != device_id:
                continue

            register_name = rule['register_name']
            if register_name not in data:
                continue

            value = data[register_name]
            if not isinstance(value, (int, float)):
                continue

            # 检查阈值
            triggered = False
            if 'threshold_high' in rule and value > rule['threshold_high']:
                triggered = True
            if 'threshold_low' in rule and value < rule['threshold_low']:
                triggered = True

            if triggered:
                self.trigger_alarm(
                    device_id=device_id,
                    register_name=register_name,
                    level=rule.get('level', 'warning'),
                    message=rule.get('message', f'{register_name} 超限'),
                    actual_value=value,
                    threshold=rule.get('threshold_high') or rule.get('threshold_low')
                )

    def trigger_alarm(self, device_id, register_name, level, message,
                      actual_value=None, threshold=None):
        """
        触发报警

        三级联动输出：
        1. 声光报警器
        2. 语音广播
        3. 前端推送
        """
        alarm_id = f"{device_id}:{register_name}"

        # 去重检查
        if not self._should_emit(alarm_id):
            return

        # 记录报警
        alarm_data = {
            'alarm_id': alarm_id,
            'device_id': device_id,
            'register_name': register_name,
            'level': level,
            'message': message,
            'actual_value': actual_value,
            'threshold': threshold,
            'timestamp': datetime.now().isoformat()
        }

        # 保存到数据库
        self.database.save_alarm(alarm_data)

        # 更新去重状态
        self._emit_history[alarm_id] = time.time()

        # 1. 声光报警器
        if self.alarm_output:
            self.alarm_output.activate(level)

        # 2. 语音广播
        if self.broadcast_system:
            self.broadcast_system.speak(message, level=level, source='alarm')

        # 3. 前端推送
        if self._websocket_emit:
            self._websocket_emit('alarm', alarm_data)

        logger.warning(f"报警触发: [{level}] {message}")

    def _should_emit(self, alarm_id):
        """检查是否应该推送（去重）"""
        if not self.dedup_config.enabled:
            return True

        last_emit = self._emit_history.get(alarm_id)
        if last_emit is None:
            return True

        elapsed = time.time() - last_emit
        return elapsed > self.dedup_config.emit_cooldown_seconds

    def acknowledge_alarm(self, alarm_id, username):
        """确认报警"""
        self._acknowledge_history[alarm_id] = time.time()
        logger.info(f"报警 {alarm_id} 已被 {username} 确认")
```

第21章：声光报警器
--------------

### 21.1 模块概述

声光报警器通过Modbus DO线圈控制报警灯塔（红/黄/绿灯）和蜂鸣器。

**硬件接线**：
```
PLC DO0 -> 红色报警灯（常亮/闪烁）
PLC DO1 -> 黄色警告灯（常亮/闪烁）
PLC DO2 -> 绿色正常灯（常亮）
PLC DO3 -> 蜂鸣器（脉冲/常响）
```

### 21.2 核心代码片段

```python
class AlarmLightPattern:
    """报警灯闪烁模式"""
    OFF = 'off'
    STEADY = 'steady'
    SLOW_FLASH = 'slow'   # 慢闪（1Hz，警告）
    FAST_FLASH = 'fast'   # 快闪（2Hz，严重）


class AlarmOutput:
    """
    工业报警输出驱动

    通过Modbus DO线圈控制报警灯塔和蜂鸣器。
    模拟模式下输出到日志。
    """

    def __init__(self, config=None):
        self.config = config or {}
        self.enabled = self.config.get('enabled', True)
        self.simulation = self.config.get('simulation', True)

        # DO线圈地址映射
        do_map = self.config.get('do_mapping', {})
        self.do_red = do_map.get('red_light', 0)
        self.do_yellow = do_map.get('yellow_light', 1)
        self.do_green = do_map.get('green_light', 2)
        self.do_buzzer = do_map.get('buzzer', 3)

        # 当前输出状态
        self.current_state = {
            'red': False,
            'yellow': False,
            'green': True,   # 默认绿灯常亮=系统正常
            'buzzer': False,
            'pattern': AlarmLightPattern.STEADY,
            'level': None,
            'message': ''
        }

    def activate(self, level, message=''):
        """
        激活报警输出

        Args:
            level: 报警级别 (critical/warning/info)
            message: 报警消息
        """
        if not self.enabled:
            return

        if level == 'critical':
            # 严重报警：红灯快闪 + 蜂鸣器
            self._write_all_do(red=True, yellow=False, green=False, buzzer=True)
            self.current_state.update({
                'red': True, 'yellow': False, 'green': False,
                'buzzer': True, 'pattern': AlarmLightPattern.FAST_FLASH,
                'level': level, 'message': message
            })

        elif level == 'warning':
            # 警告：黄灯慢闪
            self._write_all_do(red=False, yellow=True, green=False, buzzer=False)
            self.current_state.update({
                'red': False, 'yellow': True, 'green': False,
                'buzzer': False, 'pattern': AlarmLightPattern.SLOW_FLASH,
                'level': level, 'message': message
            })

        else:
            # 信息：绿灯常亮
            self._write_all_do(red=False, yellow=False, green=True, buzzer=False)
            self.current_state.update({
                'red': False, 'yellow': False, 'green': True,
                'buzzer': False, 'pattern': AlarmLightPattern.STEADY,
                'level': level, 'message': message
            })

        logger.info(f"报警输出: [{level}] {message}")

    def reset(self):
        """复位报警输出"""
        self._write_all_do(red=False, yellow=False, green=True, buzzer=False)
        self.current_state.update({
            'red': False, 'yellow': False, 'green': True,
            'buzzer': False, 'pattern': AlarmLightPattern.STEADY,
            'level': None, 'message': ''
        })

    def mute(self):
        """消音（关蜂鸣器，灯保持）"""
        self._write_do(self.do_buzzer, False)
        self.current_state['buzzer'] = False

    def _write_do(self, address, value):
        """写DO线圈（硬件/模拟双模式）"""
        if self.simulation:
            logger.debug(f"[模拟] DO{address} -> {'ON' if value else 'OFF'}")
            return True

        # 真实硬件写入
        try:
            client = self._get_modbus_client()
            if client is None:
                return False
            slave_id = self.config.get('modbus', {}).get('slave_id', 1)
            result = client.write_coil(address, value, slave=slave_id)
            return not result.isError()
        except Exception as e:
            logger.error(f"写DO{address}失败: {e}")
            return False

    def _write_all_do(self, red, yellow, green, buzzer):
        """批量写DO线圈"""
        self._write_do(self.do_red, red)
        self._write_do(self.do_yellow, yellow)
        self._write_do(self.do_green, green)
        self._write_do(self.do_buzzer, buzzer)
```

第22章：广播系统
--------------

### 22.1 模块概述

工业广播系统面向工业4.0的安全广播与语音通知。

**支持功能**：
- 语音播报/文本转语音
- MQTT主题发布（对接IP网络广播/对讲系统）
- 预设报警话术模板
- 区域定向广播

### 22.2 核心代码片段

```python
class BroadcastMessage:
    """广播消息结构体"""
    def __init__(self, text, level='info', area='all', source='system'):
        self.text = text
        self.level = level
        self.area = area
        self.source = source
        self.timestamp = datetime.now().isoformat()
        self.status = 'pending'


class BroadcastSystem:
    """
    工业广播系统

    支持两种输出通道:
    1) 本地模拟（日志/前端播放）
    2) MQTT发布到广播主题（对接IP网络广播/PA系统）
    """

    def __init__(self, config=None):
        self.config = config or {}
        self.enabled = self.config.get('enabled', True)
        self.simulation = self.config.get('simulation', True)

        self.areas = self.config.get('areas', ['all'])
        self.default_area = self.config.get('default_area', 'all')
        self.preset_templates = self.config.get('preset_templates', {})

        # 广播历史
        self._history = []
        self._max_history = 200

    def speak(self, text, level='info', area='', source='manual'):
        """
        发起广播

        Args:
            text: 广播内容
            level: 级别 info/warning/critical
            area: 广播区域
            source: 来源 manual/system/alarm
        """
        if not self.enabled:
            return {'success': False, 'message': '广播系统未启用'}

        area = area or self.default_area
        msg = BroadcastMessage(text=text, level=level, area=area, source=source)

        # 本地日志/模拟
        self._log_broadcast(msg)

        # MQTT发布（硬件对接）
        if not self.simulation:
            self._mqtt_publish(msg)

        # 记录历史
        msg.status = 'sent'
        self._history.append(msg.to_dict())
        if len(self._history) > self._max_history:
            self._history.pop(0)

        return {'success': True, 'message': '广播已发送'}

    def _log_broadcast(self, msg):
        """记录广播日志"""
        level_emoji = {'critical': '🔴', 'warning': '🟡', 'info': '🟢'}.get(msg.level, '⚪')
        logger.info(f"{level_emoji} 广播 [{msg.area}]: {msg.text}")

    def alarm_broadcast(self, alarm_data):
        """
        报警自动广播

        根据报警级别选择话术模板，自动发起广播
        """
        level = alarm_data.get('level', 'warning')
        message = alarm_data.get('message', '')
        device_id = alarm_data.get('device_id', '')

        if level == 'critical':
            template = self.preset_templates.get('alarm_critical',
                '注意！{area}发生严重报警：{message}，请立即处置！')
        else:
            template = self.preset_templates.get('alarm_warning',
                '提醒：{area}出现告警：{message}，请关注。')

        text = template.format(area='', message=message)
        self.speak(text, level=level, source='alarm')
```

### 24.1 模块概述

### 24.2 核心代码片段

```python
def init_socketio(app, database, data_collector):
    """
    初始化WebSocket

    Args:
        app: Flask应用实例
        database: 数据库实例
        data_collector: 数据采集器实例

    Returns:
        SocketIO: SocketIO实例
    """
    global socketio

    socketio = SocketIO(app, cors_allowed_origins="*", async_mode='threading')

    # 注册事件处理
    @socketio.on('connect')
    def handle_connect():
        """客户端连接"""
        logger.info(f"客户端连接: {request.sid}")
        emit('connected', {'message': '连接成功'})

    @socketio.on('disconnect')
    def handle_disconnect():
        """客户端断开"""
        logger.info(f"客户端断开: {request.sid}")

    @socketio.on('subscribe')
    def handle_subscribe(data):
        """订阅设备数据"""
        device_id = data.get('device_id')
        logger.info(f"客户端订阅设备: {device_id}")

        # 加入设备房间
        join_room(f'device_{device_id}')

        # 发送最新数据
        latest_data = database.get_latest_data(device_id)
        if latest_data:
            emit('data_update', latest_data)

    @socketio.on('unsubscribe')
    def handle_unsubscribe(data):
        """取消订阅"""
        device_id = data.get('device_id')
        logger.info(f"客户端取消订阅设备: {device_id}")

        # 离开设备房间
        leave_room(f'device_{device_id}')

    # 启动数据推送线程
    start_data_push_thread(database, data_collector)

    return socketio


def start_data_push_thread(database, data_collector):
    """
    启动数据推送线程

    注意: async_mode='threading'时用time.sleep()而非socketio.sleep()
    """
    def push_data():
        """推送数据到客户端"""
        while True:
            try:
                # 获取所有设备最新数据
                devices = database.get_device_summary()

                for device in devices:
                    device_id = device['device_id']
                    latest_data = database.get_latest_data(device_id)

                    if latest_data:
                        socketio.emit('data_update', latest_data, room=f'device_{device_id}')

                # 推送系统状态
                system_status = {
                    'timestamp': datetime.now().isoformat(),
                    'collector_stats': data_collector.get_stats(),
                    'database_stats': database.get_database_stats()
                }
                socketio.emit('system_status', system_status)

                # threading模式用time.sleep
                time.sleep(2)

            except Exception as e:
                logger.error(f"数据推送异常: {e}")
                time.sleep(5)

    thread = threading.Thread(target=push_data, daemon=True)
    thread.start()
    logger.info("数据推送线程已启动")


def emit_alarm(alarm_data):
    """发送报警通知（广播给所有客户端）"""
    if socketio:
        socketio.emit('alarm', alarm_data)


def emit_device_status(device_id, status):
    """发送设备状态更新"""
    if socketio:
        socketio.emit('device_status', {
            'device_id': device_id,
            'status': status
        }, room=f'device_{device_id}')
```

```python
class AuthManager:
    """
    用户认证管理器
    处理用户注册、登录、JWT令牌管理
    """

    def __init__(self, database):
        self.database = database
        self._init_users_table()
        self._create_default_admin()

    def _init_users_table(self):
        """初始化用户表"""
        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS users (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    username TEXT UNIQUE NOT NULL,
                    password_hash TEXT NOT NULL,
                    role TEXT NOT NULL DEFAULT 'viewer',
                    display_name TEXT,
                    email TEXT,
                    phone TEXT,
                    last_login DATETIME,
                    login_attempts INTEGER DEFAULT 0,
                    locked_until DATETIME,
                    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
                    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
                    is_active BOOLEAN DEFAULT 1
                )
            ''')

            # 创建操作日志表
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS operation_logs (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    username TEXT NOT NULL,
                    action TEXT NOT NULL,
                    target TEXT,
                    detail TEXT,
                    ip_address TEXT,
                    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
                )
            ''')

    def _create_default_admin(self):
        """创建默认管理员账户"""
        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('SELECT COUNT(*) FROM users WHERE username = ?', ('admin',))
            if cursor.fetchone()[0] == 0:
                password_hash = bcrypt.hashpw('admin123'.encode('utf-8'), bcrypt.gensalt())
                cursor.execute('''
                    INSERT INTO users (username, password_hash, role, display_name)
                    VALUES (?, ?, ?, ?)
                ''', ('admin', password_hash.decode('utf-8'), 'admin', '系统管理员'))
                logger.info("已创建默认管理员账户: admin/admin123")

    def login(self, username, password):
        """
        用户登录

        Args:
            username: 用户名
            password: 密码

        Returns:
            {'success': bool, 'token': str, 'user': dict}
        """
        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('SELECT * FROM users WHERE username = ?', (username,))
            user = cursor.fetchone()

            if user is None:
                return {'success': False, 'message': '用户名或密码错误'}

            # 检查账户是否被锁定
            if user['locked_until']:
                lock_time = datetime.fromisoformat(user['locked_until'])
                if datetime.now() < lock_time:
                    return {'success': False, 'message': f'账户已锁定，请于 {lock_time} 后重试'}

            # 验证密码
            if not bcrypt.checkpw(password.encode('utf-8'), user['password_hash'].encode('utf-8')):
                # 增加登录失败次数
                attempts = user['login_attempts'] + 1
                if attempts >= 5:
                    # 锁定账户30分钟
                    locked_until = (datetime.now() + timedelta(minutes=30)).isoformat()
                    cursor.execute('''
                        UPDATE users SET login_attempts = ?, locked_until = ?
                        WHERE username = ?
                    ''', (attempts, locked_until, username))
                    return {'success': False, 'message': '登录失败次数过多，账户已锁定30分钟'}
                else:
                    cursor.execute('''
                        UPDATE users SET login_attempts = ? WHERE username = ?
                    ''', (attempts, username))
                    return {'success': False, 'message': f'用户名或密码错误（剩余{5-attempts}次机会）'}

            # 登录成功，重置失败次数
            cursor.execute('''
                UPDATE users SET login_attempts = 0, locked_until = NULL, 
                last_login = CURRENT_TIMESTAMP WHERE username = ?
            ''', (username,))

            # 生成JWT令牌
            token = jwt.encode({
                'user_id': user['id'],
                'username': user['username'],
                'role': user['role'],
                'exp': datetime.utcnow() + timedelta(hours=JWT_EXPIRATION_HOURS)
            }, JWT_SECRET, algorithm=JWT_ALGORITHM)

            return {
                'success': True,
                'token': token,
                'user': {
                    'id': user['id'],
                    'username': user['username'],
                    'role': user['role'],
                    'display_name': user['display_name']
                }
            }

    def register(self, username, password, role='viewer', display_name=None):
        """
        注册新用户

        Args:
            username: 用户名
            password: 密码
            role: 角色
            display_name: 显示名称

        Returns:
            {'success': bool, 'message': str}
        """
        # 验证角色
        if role not in ROLES:
            return {'success': False, 'message': f'无效角色: {role}'}

        # 验证用户名
        if len(username) < 3 or len(username) > 20:
            return {'success': False, 'message': '用户名长度需在3-20之间'}

        # 验证密码强度
        if len(password) < 6:
            return {'success': False, 'message': '密码长度至少6位'}

        # 加密密码
        password_hash = bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt())

        with self.database.get_connection() as conn:
            cursor = conn.cursor()
            try:
                cursor.execute('''
                    INSERT INTO users (username, password_hash, role, display_name)
                    VALUES (?, ?, ?, ?)
                ''', (username, password_hash.decode('utf-8'), role, display_name or username))
                return {'success': True, 'message': '注册成功'}
            except sqlite3.IntegrityError:
                return {'success': False, 'message': '用户名已存在'}


def jwt_required(f):
    """JWT认证装饰器"""
    @wraps(f)
    def decorated(*args, **kwargs):
        token = request.headers.get('Authorization', '').replace('Bearer ', '')
        if not token:
            return jsonify({'error': '缺少认证令牌'}), 401

        try:
            payload = jwt.decode(token, JWT_SECRET, algorithms=[JWT_ALGORITHM])
            request.current_user = payload
        except jwt.ExpiredSignatureError:
            return jsonify({'error': '令牌已过期'}), 401
        except jwt.InvalidTokenError:
            return jsonify({'error': '无效的令牌'}), 401

        return f(*args, **kwargs)
    return decorated


def role_required(*roles):
    """角色权限装饰器"""
    def decorator(f):
        @wraps(f)
        def decorated(*args, **kwargs):
            if not hasattr(request, 'current_user'):
                return jsonify({'error': '未认证'}), 401

            user_role = request.current_user.get('role')
            if user_role not in roles:
                return jsonify({'error': '权限不足'}), 403

            return f(*args, **kwargs)
        return decorated
    return decorator
```

```python
class EventBus:
    """
    事件总线

    实现发布-订阅模式，模块间通过事件通信，降低耦合度。

    使用示例：
        event_bus = EventBus()
        event_bus.subscribe('data_collected', on_data_collected)
        event_bus.publish('data_collected', {'device_id': 'plc01', 'data': {...}})
    """

    def __init__(self):
        self._subscribers = defaultdict(list)
        self._lock = threading.Lock()

    def subscribe(self, event_type, callback):
        """订阅事件"""
        with self._lock:
            self._subscribers[event_type].append(callback)

    def unsubscribe(self, event_type, callback):
        """取消订阅"""
        with self._lock:
            if event_type in self._subscribers:
                self._subscribers[event_type].remove(callback)

    def publish(self, event_type, data=None):
        """发布事件"""
        with self._lock:
            callbacks = list(self._subscribers.get(event_type, []))

        for callback in callbacks:
            try:
                callback(data)
            except Exception as e:
                logger.error(f"事件处理异常 [{event_type}]: {e}")


class ModuleRegistry:
    """
    模块注册表

    管理所有模块的注册、初始化、启动、停止。
    """

    def __init__(self):
        self._modules = {}
        self._order = []  # 启动顺序

    def register(self, name, module, priority=100):
        """注册模块"""
        self._modules[name] = {
            'instance': module,
            'priority': priority,
            'status': 'registered'
        }
        self._order.append(name)
        logger.info(f"模块注册: {name}")

    def get(self, name):
        """获取模块实例"""
        return self._modules.get(name, {}).get('instance')

    def start_all(self):
        """按优先级启动所有模块"""
        sorted_modules = sorted(
            self._modules.items(),
            key=lambda x: x[1]['priority']
        )

        for name, info in sorted_modules:
            module = info['instance']
            if hasattr(module, 'start'):
                try:
                    module.start()
                    info['status'] = 'running'
                    logger.info(f"模块启动: {name}")
                except Exception as e:
                    info['status'] = 'error'
                    logger.error(f"模块启动失败 [{name}]: {e}")

    def stop_all(self):
        """停止所有模块"""
        for name, info in reversed(self._modules.items()):
            module = info['instance']
            if hasattr(module, 'stop'):
                try:
                    module.stop()
                    info['status'] = 'stopped'
                    logger.info(f"模块停止: {name}")
                except Exception as e:
                    logger.error(f"模块停止失败 [{name}]: {e}")

    def get_status(self):
        """获取所有模块状态"""
        return {
            name: {
                'status': info['status'],
                'priority': info['priority']
            }
            for name, info in self._modules.items()
        }


class HealthChecker:
    """
    健康检查器

    定期检查各模块的运行状态，发现异常及时报警。
    """

    def __init__(self, module_registry, alarm_manager=None):
        self.module_registry = module_registry
        self.alarm_manager = alarm_manager
        self._running = False
        self._thread = None

    def start(self, interval=30):
        """启动健康检查"""
        self._running = True
        self._thread = threading.Thread(
            target=self._check_loop,
            args=(interval,),
            daemon=True
        )
        self._thread.start()
        logger.info(f"健康检查器已启动，间隔 {interval} 秒")

    def stop(self):
        """停止健康检查"""
        self._running = False
        if self._thread:
            self._thread.join(timeout=5)

    def _check_loop(self, interval):
        """检查主循环"""
        while self._running:
            try:
                self._check_all_modules()
            except Exception as e:
                logger.error(f"健康检查异常: {e}")
            time.sleep(interval)

    def _check_all_modules(self):
        """检查所有模块"""
        status = self.module_registry.get_status()

        for name, info in status.items():
            if info['status'] == 'error':

                        device_id='SYSTEM',

                        message=f'模块 {name} 状态异常'
                    )
```

```python
class BaseGateway:
    """
    网关基类

    定义网关的通用接口：
    - connect(): 连接设备
    - disconnect(): 断开连接
    - read_data(): 读取数据
    - write_data(): 写入数据
    """

    def __init__(self, config):
        self.config = config
        self.connected = False
        self._lock = threading.Lock()

    def connect(self):
        """连接设备"""
        raise NotImplementedError

    def disconnect(self):
        """断开连接"""
        raise NotImplementedError

    def read_data(self, address, length):
        """读取数据"""
        raise NotImplementedError

    def write_data(self, address, value):
        """写入数据"""
        raise NotImplementedError

    def is_connected(self):
        """检查连接状态"""
        return self.connected


class ModbusGateway(BaseGateway):
    """
    Modbus协议网关

    支持Modbus TCP和Modbus RTU。
    """

    def __init__(self, config):
        super().__init__(config)
        self.host = config.get('host', '192.168.1.100')
        self.port = config.get('port', 502)
        self.slave_id = config.get('slave_id', 1)
        self.client = None

    def connect(self):
        """连接Modbus设备"""
        try:
            from pymodbus.client import ModbusTcpClient
            self.client = ModbusTcpClient(self.host, port=self.port)
            self.connected = self.client.connect()
            if self.connected:
                logger.info(f"Modbus连接成功: {self.host}:{self.port}")
            else:
                logger.error(f"Modbus连接失败: {self.host}:{self.port}")
            return self.connected
        except Exception as e:
            logger.error(f"Modbus连接异常: {e}")
            return False

    def disconnect(self):
        """断开Modbus连接"""
        if self.client:
            self.client.close()
            self.connected = False
            logger.info("Modbus连接已断开")

    def read_data(self, address, length=1):
        """读取保持寄存器"""
        if not self.connected:
            return None

        try:
            result = self.client.read_holding_registers(
                address, length, slave=self.slave_id
            )
            if result.isError():
                logger.error(f"Modbus读取失败: {result}")
                return None
            return result.registers
        except Exception as e:
            logger.error(f"Modbus读取异常: {e}")
            return None

    def write_data(self, address, value):
        """写入单个寄存器"""
        if not self.connected:
            return False

        try:
            result = self.client.write_register(
                address, value, slave=self.slave_id
            )
            return not result.isError()
        except Exception as e:
            logger.error(f"Modbus写入异常: {e}")
            return False


class MQTTSubscriber(BaseGateway):
    """
    MQTT订阅器

    订阅MQTT主题，接收设备推送的数据。
    """

    def __init__(self, config):
        super().__init__(config)
        self.broker = config.get('broker', 'localhost')
        self.port = config.get('port', 1883)
        self.topic = config.get('topic', '#')
        self.client = None
        self._callbacks = []

    def connect(self):
        """连接MQTT Broker"""
        try:
            import paho.mqtt.client as mqtt
            self.client = mqtt.Client()
            self.client.on_connect = self._on_connect
            self.client.on_message = self._on_message
            self.client.connect(self.broker, self.port)
            self.client.loop_start()
            self.connected = True
            logger.info(f"MQTT连接成功: {self.broker}:{self.port}")
            return True
        except Exception as e:
            logger.error(f"MQTT连接异常: {e}")
            return False

    def disconnect(self):
        """断开MQTT连接"""
        if self.client:
            self.client.loop_stop()
            self.client.disconnect()
            self.connected = False

    def subscribe(self, topic, callback):
        """订阅主题"""
        self._callbacks.append(callback)
        if self.client:
            self.client.subscribe(topic)

    def _on_connect(self, client, userdata, flags, rc):
        """连接回调"""
        logger.info(f"MQTT已连接，订阅主题: {self.topic}")
        client.subscribe(self.topic)

    def _on_message(self, client, userdata, msg):
        """消息回调"""
        try:
            payload = json.loads(msg.payload.decode('utf-8'))
            for callback in self._callbacks:
                callback(msg.topic, payload)
        except Exception as e:
            logger.error(f"MQTT消息处理异常: {e}")
```

```python
class TSDBAdapter:
    """
    TDengine智能层适配器

    功能：
    1. 定期从TDengine读取遥测数据
    2. 喂入OEE/预测性维护/SPC/能源模块
    3. 将计算结果写回TDengine
    """

    def __init__(self, tdengine, oee_calculator=None,
                 predictive_maintenance=None, spc_analyzer=None,
                 energy_manager=None, config=None):
        self.tdengine = tdengine
        self.oee_calculator = oee_calculator
        self.predictive_maintenance = predictive_maintenance
        self.spc_analyzer = spc_analyzer
        self.energy_manager = energy_manager
        self.config = config or {}

        # 数据同步间隔（秒）
        self.sync_interval = self.config.get('sync_interval', 10)

        # 结果写入间隔（秒）
        self.result_interval = self.config.get('result_interval', 60)

        # 运行状态
        self._running = False

        # 统计信息
        self.stats = {
            'sync_cycles': 0,
            'records_synced': 0,
            'results_written': 0,
            'errors': 0
        }

    def register_device(self, device_id, registers):
        """
        注册设备及其寄存器

        Args:
            device_id: 设备ID
            registers: 寄存器列表 [{'name': 'temperature', 'unit': '°C'}, ...]
        """
        self._registered_devices[device_id] = {
            'registers': registers,
            'last_sync': None
        }
        logger.info(f"注册设备 {device_id}，{len(registers)} 个寄存器")

    def _sync_loop(self):
        """数据同步主循环"""
        while self._running:
            try:
                self._sync_data()
                self.stats['sync_cycles'] += 1
            except Exception as e:
                logger.error(f"数据同步异常: {e}")
                self.stats['errors'] += 1
            time.sleep(self.sync_interval)

    def _sync_data(self):
        """同步数据到智能层模块"""
        for device_id, info in self._registered_devices.items():
            try:
                # 从TDengine读取最新数据
                data = self.tdengine.query_latest(device_id)
                if not data:
                    continue

                # 喂入各智能层模块
                for register_name, value in data.items():
                    # 预测性维护
                    if self.predictive_maintenance:
                        self.predictive_maintenance.feed_data(
                            device_id, register_name, value
                        )

                    # SPC分析器
                    if self.spc_analyzer:
                        self.spc_analyzer.feed_data(
                            device_id, register_name, value
                        )

                self.stats['records_synced'] += len(data)

            except Exception as e:
                logger.error(f"设备 {device_id} 数据同步失败: {e}")

    def _result_loop(self):
        """结果写入主循环"""
        while self._running:
            try:
                self._write_results()
            except Exception as e:
                logger.error(f"结果写入异常: {e}")
            time.sleep(self.result_interval)

    def _write_results(self):
        """将智能层计算结果写回TDengine"""
        # OEE结果
        if self.oee_calculator:
            for device_id in self._registered_devices:
                oee = self.oee_calculator.calculate_oee(device_id)
                if oee:
                    self.tdengine.write_oee(device_id, oee)
                    self.stats['results_written'] += 1

        # 健康评分
        if self.predictive_maintenance:
            for device_id in self._registered_devices:
                health = self.predictive_maintenance.calculate_health_score(device_id)
                if health:
                    self.tdengine.write_health(device_id, health)
                    self.stats['results_written'] += 1
```

### 33.1 常见问题

**问题1：设备连接失败**
```
原因：网络不通、IP地址错误、端口被占用
解决：
1. 检查网络连接：ping 192.168.1.53
2. 检查端口是否开放：telnet 192.168.1.53 502
3. 检查防火墙设置
```

**问题2：数据采集失败**
```
原因：设备断开、寄存器地址错误、数据类型不匹配
解决：
1. 检查设备连接状态
2. 验证寄存器地址
3. 检查数据类型配置
```

**问题3：报警不触发**
```
原因：报警规则未启用、阈值设置错误、报警管理器未启动
解决：
1. 检查报警规则配置
2. 验证阈值设置
3. 检查报警管理器状态
```

**问题4：Web页面无法访问**
```
原因：端口被占用、Flask应用未启动、防火墙阻止
解决：
1. 检查端口占用：netstat -ano | findstr 5000
2. 检查Flask应用日志
3. 检查防火墙设置
```

**问题5：数据库锁定**
```
原因：多个进程同时写入、事务未提交
解决：
1. 检查是否有其他进程占用数据库
2. 使用WAL模式提升并发性能
3. 设置合理的超时时间
```

### 附录C：API接口列表

**设备管理API**：
- `GET /api/devices` - 获取设备列表
- `POST /api/devices` - 添加设备
- `PUT /api/devices/<id>` - 更新设备
- `DELETE /api/devices/<id>` - 删除设备
- `POST /api/devices/<id>/stop` - 停止设备
- `POST /api/devices/<id>/start` - 启动设备

**报警API**：
- `GET /api/alarms` - 获取报警列表
- `POST /api/alarms/<id>/acknowledge` - 确认报警
- `GET /api/alarms/kpi` - 获取报警KPI

**控制API**：
- `POST /api/control/batch` - 批量控制
- `POST /api/control/estop` - 紧急停机
- `POST /api/control/reset-estop` - 重置紧急停机

**数据API**：
- `GET /api/data/realtime` - 获取实时数据
- `GET /api/data/history` - 获取历史数据
- `GET /api/data/export` - 导出数据

**工业4.0 API**：
- `GET /api/industry40/oee` - 获取OEE数据
- `GET /api/industry40/spc` - 获取SPC数据
- `GET /api/industry40/predictive` - 获取预测性维护数据
- `GET /api/industry40/energy` - 获取能源数据

**认证API**：
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/refresh` - 刷新令牌
