# 基于Python的工业数据采集与监控系统

## 项目简介

本系统是一个基于Python的工业数据采集与监控系统（SCADA），实现对工业设备的实时数据采集、存储、可视化和报警管理。

## 功能模块

### 1. 数据采集模块
- Modbus TCP/RTU协议支持
- 多设备并发采集
- 采集频率可配置
- 数据校验和错误处理

### 2. 数据存储模块
- SQLite数据库存储
- 历史数据归档
- 数据压缩和清理
- CSV/Excel导出

### 3. Web可视化模块
- 实时数据Dashboard
- 历史趋势图表
- 设备状态监控
- 响应式设计

### 4. 报警管理模块
- 阈值报警配置
- 报警记录和查询
- 报警通知（邮件/短信）
- 报警统计分析

## 技术栈

- **后端**: Python 3.12 + Flask
- **数据库**: SQLite
- **前端**: HTML5 + CSS3 + JavaScript
- **图表**: ECharts
- **通信**: pymodbus (Modbus协议)
- **实时**: WebSocket (Flask-SocketIO)

## 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置设备
编辑 `配置/devices.yaml`，配置Modbus设备参数。

### 3. 启动系统
```bash
python run.py
```

### 4. 访问系统
打开浏览器访问 `http://localhost:5000`
