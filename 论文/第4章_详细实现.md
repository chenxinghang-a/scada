## 4.1 采集层实现

### 4.1.1 Modbus客户端实现

Modbus客户端模块基于pymodbus库实现，封装了Modbus TCP和Modbus RTU两种通信方式。客户端类提供了统一的读取接口，支持多种数据类型的解码。

核心代码如下：

```python
from pymodbus.client import ModbusTcpClient
import struct

class ModbusClient:
    def __init__(self, config):
        self.host = config.get('host')
        self.port = config.get('port', 502)
        self.slave_id = config.get('slave_id', 1)
        self.client = None
        self.connected = False
    
    def connect(self):
        self.client = ModbusTcpClient(self.host, port=self.port)
        self.connected = self.client.connect()
        return self.connected
    
    def read_holding_registers(self, address, count):
        result = self.client.read_holding_registers(
            address=address, count=count, slave=self.slave_id
        )
        if result.isError():
            return None
        return result.registers
    
    def decode_float32(self, registers):
        raw = struct.pack('>HH', registers[0], registers[1])
        return struct.unpack('>f', raw)[0]
```

### 4.1.2 设备管理器实现

设备管理器负责管理所有设备的配置和连接，支持从YAML配置文件加载设备配置，支持模拟模式和真实模式的切换。

```python
class DeviceManager:
    def __init__(self, config_path, simulation_mode=True):
        self.config_path = config_path
        self.simulation_mode = simulation_mode
        self.devices = {}
        self.clients = {}
        self.load_config()
    
    def get_client(self, device_id):
        if device_id not in self.clients:
            device_config = self.devices.get(device_id)
            if self.simulation_mode:
                client = SimulatedModbusClient(device_config)
            else:
                client = ModbusClient(device_config)
            self.clients[device_id] = client
        return self.clients[device_id]
```

### 4.1.3 数据采集器实现

数据采集器负责定时轮询设备并获取数据，采用多线程设计，每个设备使用独立的定时器进行数据采集。

```python
class DataCollector:
    def __init__(self, device_manager, database, alarm_manager):
        self.device_manager = device_manager
        self.database = database
        self.alarm_manager = alarm_manager
        self.data_queue = Queue(maxsize=10000)
        self.running = False
    
    def _collect_device_data(self, device_id, device_config):
        client = self.device_manager.get_client(device_id)
        registers = device_config.get('registers', [])
        
        for register in registers:
            raw_values = client.read_holding_registers(
                register['address'], register.get('length', 1)
            )
            value = self._decode_value(raw_values, register)
            value = value * register.get('scale', 1) + register.get('offset', 0)
            
            self.data_queue.put({
                'device_id': device_id,
                'register_name': register['name'],
                'value': value,
                'unit': register.get('unit', '')
            })
```

## 4.2 存储层实现

### 4.2.1 数据库初始化

数据库模块基于SQLite实现，初始化时自动创建四张数据表和相关索引。

```python
class Database:
    def __init__(self, db_path):
        self.db_path = db_path
        Path(db_path).parent.mkdir(parents=True, exist_ok=True)
        self._init_database()
    
    def _init_database(self):
        with self.get_connection() as conn:
            cursor = conn.cursor()
            cursor.execute('''
                CREATE TABLE IF NOT EXISTS realtime_data (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    device_id TEXT NOT NULL,
                    register_name TEXT NOT NULL,
                    value REAL,
                    unit TEXT,
                    timestamp DATETIME NOT NULL
                )
            ''')
            # 创建索引
            cursor.execute('''
                CREATE INDEX IF NOT EXISTS idx_realtime_device_time 
                ON realtime_data(device_id, timestamp)
            ''')
```

### 4.2.3 数据查询接口

数据查询接口支持多种查询方式，包括实时数据查询、历史数据查询和数据聚合查询。

```python
def get_history_data(self, device_id, register_name, 
                     start_time, end_time, interval='1min'):
    if interval == '1min':
        group_format = '%Y-%m-%d %H:%M:00'
    elif interval == '1hour':
        group_format = '%Y-%m-%d %H:00:00'
    
    cursor.execute('''
        SELECT strftime(?, timestamp) as time_bucket,
               AVG(value) as avg_value,
               MIN(value) as min_value,
               MAX(value) as max_value,
               COUNT(*) as sample_count
        FROM history_data
        WHERE device_id = ? AND register_name = ?
            AND timestamp BETWEEN ? AND ?
        GROUP BY time_bucket
        ORDER BY time_bucket
    ''', (group_format, device_id, register_name, start_time, end_time))
```

### 4.2.4 数据导出功能

数据导出模块支持将设备数据和报警记录导出为CSV格式。

```python
class DataExport:
    def export_device_data(self, database, device_id, 
                           start_time, end_time, format='csv'):
        registers = database.get_device_registers(device_id)
        data = []
        for register_name in registers:
            register_data = database.get_history_data(
                device_id, register_name, start_time, end_time
            )
            data.extend(register_data)
        
        if format == 'csv':
            filepath = f'exports/{device_id}_{datetime.now():%Y%m%d_%H%M%S}.csv'
            with open(filepath, 'w', encoding='utf-8-sig') as f:
                writer = csv.DictWriter(f, fieldnames=data[0].keys())
                writer.writeheader()
                writer.writerows(data)
            return filepath
```

### 4.3.4 报警引擎实现

报警引擎负责检测数据是否超出阈值，并触发相应的报警。

```python
class AlarmManager:
    def check_alarm(self, device_id, register_name, value, timestamp):
        for rule_id, rule_config in self.rules.items():
            if not rule_config.get('enabled', True):
                continue
            
            if (rule_config['device_id'] == device_id and
                rule_config['register_name'] == register_name):
                
                triggered = self._check_condition(
                    value, rule_config['condition'], rule_config['threshold']
                )
                
                if triggered:
                    self._trigger_alarm(rule_config, device_id, 
                                       register_name, value, timestamp)
    
    def _check_condition(self, value, condition, threshold):
        if condition == 'greater_than':
            return value > threshold
        elif condition == 'less_than':
            return value < threshold
```

## 4.5 关键算法实现

### 4.5.1 数据类型转换算法

Modbus寄存器为16位无符号整数，需要根据数据类型进行转换。本系统支持的数据类型转换算法如下：

```python
def decode_float32(self, registers):
    """将两个16位寄存器转换为32位浮点数"""
    raw = struct.pack('>HH', registers[0], registers[1])
    return struct.unpack('>f', raw)[0]

def decode_uint16(self, register):
    """无符号16位整数"""
    return register & 0xFFFF

def decode_int16(self, register):
    """有符号16位整数"""
    if register & 0x8000:
        return register - 0x10000
    return register
```

### 4.5.2 缩放系数计算

工业现场的物理量需要通过缩放系数转换为实际值：

```python
value = raw_value * scale + offset
```

其中，raw_value为原始寄存器值，scale为缩放系数，offset为偏移量。

### 4.5.3 报警阈值判断算法

报警阈值判断支持多种条件类型：

```python
def _check_condition(self, value, condition, threshold):
    if condition == 'greater_than':
        return value > threshold
    elif condition == 'less_than':
        return value < threshold
    elif condition == 'equal_to':
        return abs(value - threshold) < 0.0001
    elif condition == 'greater_equal':
        return value >= threshold
    elif condition == 'less_equal':
        return value <= threshold
```
